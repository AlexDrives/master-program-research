# 杜克大学 - Computer Science MS 课程介绍

> 官网核验日期：2026-07-06  
> 项目：M.S. in Computer Science

---

## 项目概况

- Duke CS MS 有 course-only 与 project/thesis 两种 option。
- 两种 option 都要求 30 graduate credits，并通过 master's exam。
- 可选 concentration 包括 Artificial Intelligence/Machine Learning 和 Cybersecurity；本文件完整保留两类 concentration 的官方课程池。

## 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | 至少 30 graduate credits |
| Residency | full-time registered for three semesters |
| Master's exam | 必须完成；course-only 为 portfolio oral exam；project/thesis 为公开 defense |
| Course-only | 至少 18 credits graduate CS；至少 6 credits outside CS；至少 6 credits approved electives；完成 CS 704 |
| Project/Thesis | 至少 12 credits graduate CS；至少 6 credits outside CS；至少 6 credits approved electives；最多 6 credits ungraded research；完成 CS 704 |
| 成绩 | 除 ungraded research 外，计入学位课程需 B- or higher |
| Project/Thesis adviser | 需 faculty adviser、topic description、supervisory committee |

## 官方课程结构 / 全部课程池

### 1. General MS course structure

| 课程池 | Course-only | Project/Thesis |
|---|---:|---:|
| Graduate CS coursework | 至少 18 credits | 至少 12 credits |
| Outside-CS coursework | 至少 6 credits | 至少 6 credits |
| Approved electives | 至少 6 credits | 至少 6 credits |
| Ungraded research | 不适用 | 最多 6 credits |
| CS 704 | 必修 | 必修 |

### 2. Artificial Intelligence / Machine Learning concentration

| 模块 | 官方课程池 |
|---|---|
| Breadth | CompSci 570 Artificial Intelligence 或 CompSci 671 |
| Depth | CompSci 671 Theory and Algorithms for ML；CompSci 590.XX Machine Learning Algorithms；CompSci 590.XX Reinforcement Learning；CompSci 527 Computer Vision；CompSci 571 Probabilistic ML；CompSci 572 Introduction to Natural Language Processing；CompSci 574 Deep Learning Fundamentals |
| Elective | CompSci 555 Probability and Statistics for ECE；CompSci 590 Topics in Computational Biology；CompSci 561 Computational Sequence Biology；CompSci 590 Cryo-EM Analysis；CompSci 590 Computational Economics；CompSci 590 Generative Models；CompSci 675 Introduction to Deep Learning；ECE 590 Advanced Topics in Deep Learning；ECE 661 Computer Engineering Machine Learning and Deep Neural Networks；ECE 563 Machine Learning in Adversarial Settings |
| Project/Thesis concentration route | project/thesis 必须与 concentration 相关；另需任意两门 breadth/depth/elective 课程，其中至少一门为 depth |

### 3. Cybersecurity concentration

| 模块 | 官方课程池 |
|---|---|
| Breadth | CompSci 581 Graduate Computer Security；也可用 CompSci 510 或 CompSci 514 |
| Depth | CompSci 581 Graduate Computer Security；CompSci 590 Secure Software Systems；CompSci 590 CryptoCurrency；CompSci 590 Applied Cryptography；CompSci 590 Blockchain；CompSci 590 Privacy and Fairness；CompSci 590 Cloud Security；CompSci 582 Cryptography |
| Elective | CompSci 516 Database Systems；CompSci 555 Probability and Statistics for ECE；CompSci 554 Fault Tolerant and Testable Computer Systems；CompSci 590 Computer Architecture and Hardware Acceleration；CompSci 564 Edge Computing；CompSci 590 Coding Theory；CompSci 630 Randomized Algorithms；CompSci 650 Advanced Computer Architecture；ECE 663 Machine Learning in Adversarial Settings |
| Portfolio | concentration 还需 portfolio，至少列出两项与 concentration 相关的活动 |

## 官方课程描述抓手

- Duke MS Requirements 页面是 general requirements 与 optional concentration course pools 的主来源。
- 逐门课程描述使用 Duke CS Course Catalog / Course Bulletin；CompSci 590.XX 为 topics course，题名和编号后缀会随学期变化。

## AI / 机器人 / 感知相关课程标注

- 最相关 concentration：Artificial Intelligence/Machine Learning。
- 感知相关：CompSci 527 Computer Vision。
- 机器人“大脑”相关：CompSci 570、671、571、572、574、590 Reinforcement Learning、590 Machine Learning Algorithms、590 Generative Models。
