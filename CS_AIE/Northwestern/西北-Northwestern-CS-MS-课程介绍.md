# 西北大学 - Computer Science MS 课程介绍

> 官网核验日期：2026-07-06  
> 项目：M.S. in Computer Science

---

## 项目概况

- Northwestern CS MS 要求 12 units graduate-level credits。
- 学生可走 course-only、project、thesis 三种 plan；course-only 是默认 plan。
- 课程可包括 CS courses、经批准的其他院系相关课程和 research。

## 修读要求

| Plan | 官方要求 |
|---|---|
| Plan C Course-only | 12 courses / units，经 adviser 批准，构成 coherent program of study |
| Plan B Project | 12 graduate-level units；可含最多 2 units COMP_SCI 590 research |
| Plan A Thesis | 12 graduate-level units；可含最多 3 units COMP_SCI 590 research |
| 课程层级 | Graduate-level；实际课程选择由 adviser / MS program approval 管理 |
| 时长 | 官网项目页说明通常 3-4 quarters |

## 官方课程结构 / 全部课程池

### 1. Degree plans

| 模块 | 说明 |
|---|---|
| CS graduate courses | 主体课程池；需形成 coherent program |
| Approved non-CS graduate courses | 可来自相关院系，需提前批准 |
| COMP_SCI 590 Research | Thesis plan 最多 3 units；Project plan 最多 2 units |
| Course-only approved courses | 12 门经 adviser 批准的课程 |

### 2. 官方 catalog / project-thesis 结构

| Plan | Completion component |
|---|---|
| Thesis | 与 CS faculty 做正式 thesis research |
| Project | multi-quarter application project |
| Course-only | 12 approved courses |

### 3. 官方研究 / 课程方向池

Northwestern CS 官网和 TGS 页面列出的方向池包括：

| Area | 说明 |
|---|---|
| Artificial Intelligence | reasoning, planning, AI |
| Machine Learning | ML, statistical learning |
| Human-Computer Interaction | HCI |
| Distributed Interactive Systems | interactive systems |
| Theoretical Computer Science | theory |
| Systems and Networking | systems, distributed systems, networking |
| Graphics / Visualization / Vision | visual computing |
| Robotics | robotics / autonomous or interactive systems |
| Databases / Data Science | data systems |
| Security / Privacy | security |
| Software Engineering / Programming Languages | software and PL |

## 官方课程描述抓手

- Northwestern catalog 的 CS MS 页面定义 plan 和 12 units；McCormick CS MS 页面提供项目、thesis/project 信息和 courses 入口。
- 具体逐门课程以 Northwestern course catalog / class schedule 和 adviser approval 为准；项目页没有固定“所有学生必须从某张 elective list 选择”的清单。

## AI / 机器人 / 感知相关课程标注

- 最相关方向：AI, ML, robotics, vision/graphics/visualization, HCI, systems。
- COMP_SCI 590 可用于 thesis/project research，但有 plan-specific 上限。
