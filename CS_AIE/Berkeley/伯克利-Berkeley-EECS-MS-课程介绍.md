# 加州大学伯克利分校 - EECS MS 课程介绍

> 官网核验日期：2026-07-06  
> 项目：M.S. in Electrical Engineering and Computer Sciences

---

## 项目概况

- Berkeley EECS 的研究型 MS 通常与 PhD / MS-PhD 研究培养体系相连，不应理解为普通授课型 terminal CS master。
- EECS 官方 research programs 页面按研究方向组织；MS Student Guide 给出 MS 学位的核心 unit / research 规则。
- 本文件保留完整官方学位结构和研究方向池；AI / robotics / perception 仅作为相关方向单独标注。

## 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | Minimum 24 units |
| Graduate EECS coursework | 至少 10 units 的 200-series EE/CS 课程，排除 298/299 |
| Research units | Plan I: 4-10 units of 299；Plan II: 3-6 units of 299 |
| 剩余课程 | 可为 100- 或 200-series；不必全部是 EECS；299 不计入普通课程部分 |
| Grading | 除 299 通常 S/U 外，课程需 letter grade |
| College of Engineering 限制 | 298、300/301、601/602 不计入 master minimum unit；299 在 graduate coursework in major 中最多有单位上限 |
| 5th Year MS 补充 | 5th Year MS 页面另列 technical courses、research project、breadth courses 三部分 |

## 官方课程结构 / 全部课程池

### 1. MS Student Guide 的课程结构

| 课程池 | 要求 |
|---|---|
| 200-series EE/CS technical courses | 至少 10 units，排除 EE/CS 298、299 |
| EE/CS 299 research | Plan I 4-10 units；Plan II 3-6 units |
| 100/200-series remainder | 补足 24 units，可含非 EECS 课程，需 advisor/项目批准 |

### 2. 5th Year MS 官方结构

| 模块 | 要求 |
|---|---|
| Technical courses | 至少 10 units 的 200-level EE or CS，排除 298/299，并由 5th year MS faculty advisor 批准 |
| Research project | 与 faculty advisor 合作完成研究项目 |
| Breadth courses | 8-11 units 的 EECS 外 100/200-level 课程；至少 8 units outside EECS，课程应构成 coherent set，并经 advisor 批准 |

### 3. EECS research programs 官方方向池

Berkeley EECS 官方 graduate research programs 页面按研究生态组织，课程计划通常围绕 advisor 和 research area 形成。公开页面可核验的方向池包括：

| Research area / program | 说明 |
|---|---|
| Artificial Intelligence | AI、reasoning、planning、intelligent systems |
| Machine Learning | ML、deep learning、statistical learning |
| Robotics | embodied systems, manipulation, autonomy |
| Computer Vision and Graphics | visual perception, graphics, computational imaging |
| Control, Intelligent Systems, and Robotics | control / robotics / autonomy 交叉 |
| Database Management Systems | databases, data systems |
| Education | CS education |
| Energy | energy systems and computing |
| Human-Computer Interaction | HCI, interaction |
| Integrated Circuits | circuits |
| Micro/Nano Electro Mechanical Systems | MEMS/NEMS |
| Operating Systems and Networking | OS, networking, distributed systems |
| Programming Systems | PL, software systems |
| Scientific Computing | numerical / scientific computing |
| Security | security and privacy |
| Signal Processing | signal processing |
| Theory | algorithms and theoretical CS |

## 官方课程描述抓手

- 学位要求以 MS Student Guide、College of Engineering Graduate Guide 和 EECS advising 为准。
- 逐门课程描述应从 Berkeley Academic Guide / EECS course catalog 查询；研究型 MS 没有在项目页公开一个固定、全员相同的 elective list。
- 对 5th Year MS，technical courses 与 breadth courses 均需 advisor 批准，因此官方课程池是“符合编号/部门/批准规则的课程”，不是固定菜单。

## AI / 机器人 / 感知相关课程标注

- 相关官方方向：Artificial Intelligence, Machine Learning, Robotics, Computer Vision and Graphics, Control/Intelligent Systems/Robotics。
- 相关课程计划通常应围绕 200-level EECS technical courses、EE/CS 299 research 和 faculty advisor 研究方向组合。
