# 马里兰大学 College Park - Computer Science MS 课程介绍

> 官网核验日期：2026-07-06  
> 项目：M.S. in Computer Science

---

## 项目概况

- UMD CS MS 有 thesis 与 non-thesis 两种 completion option。
- Graduate catalog 说明：CSMS MS degree 不开放给 PhD 在读学生；另有 PhD coursework 规则，不应混淆。
- 项目核心为 30 credits、CMSC 主体课程和 600-800 level qualifying coursework。

## 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | 30 credits |
| CMSC credits | 至少 21 credits in CMSC / computer science courses |
| 高阶课程 | 至少 18 credits at 600-800 level |
| Qualifying coursework | 至少 4 门 MS qualifying CS courses，600-800 level，覆盖 8 个 areas 中的 4 个 |
| 成绩 | 至少 2 门 qualifying courses 为 A range；另外 2 门 B- or above；总体 B average |
| Thesis option | 24 credits coursework + 6 credits CMSC 799 Master's Thesis Research；需 thesis / defense |
| Non-thesis option | 30 credits coursework、comprehensive examinations、scholarly paper；最多 6 credits CMSC 798 可计入 |
| 外系课程 | 需 advisor / Graduate Office 预先批准 |
| Transfer | 通常最多 6 credits 可转入，需批准 |

## 官方课程结构 / 全部课程池

### 1. Degree course structure

| 模块 | 学分 / 要求 |
|---|---|
| Core requirements | 至少 4 门 600-800 level MS qualifying CS courses，覆盖 4/8 areas；总计至少 21 CMSC credits |
| Thesis option | CMSC 799 Master's Thesis Research，6 credits |
| Non-thesis option | Scholarly paper + 6 additional credits coursework；可含 CMSC 798 independent research；CMSC 898 仅 PhD 可用 |

### 2. 官方研究 / area pool

UMD CS catalog 与 graduate pages 列出的研究和课程生态覆盖：

| Area | 说明 |
|---|---|
| Artificial Intelligence | AI, planning, reasoning |
| Computer Systems and Networking | systems, networking |
| Database Systems | database systems |
| Programming Languages | PL |
| Software Engineering | software engineering |
| Scientific Computing | scientific computing |
| Algorithms and Computation Theory | algorithms, theory |
| Computer Vision | vision, image understanding |
| Computer Audition | audio perception |
| Geometric Computing | geometry |
| Graphics | graphics |
| Human-Computer Interaction | HCI |
| Bioinformatics | computational biology / bioinformatics |

### 3. Qualifying course pool

- 官方 MS policy 要求“8 个 areas 中覆盖 4 个”的 MS qualifying computer science courses。
- 逐门 qualifying course designation 由 graduate office / graduate coursework page 给出，并会随开课变化；400-level 即使有 field designation，也不能计入 qualifying coursework。
- 因此本文件不删除任何非 AI 领域：所有 600-800 level qualifying areas 都可用于满足 MS breadth/qualifying 结构。

## AI / 机器人 / 感知相关课程标注

- 最相关 area：Artificial Intelligence, Computer Vision, Computer Audition, Graphics, HCI, Systems。
- 可通过 CMSC 798 / CMSC 799 落地 research / thesis；但是否计入学位需遵守 thesis / non-thesis 上限。

## 官方课程描述抓手

- 逐门课程描述以 UMD Graduate Coursework、Schedule of Classes 和 Graduate Office 当年 qualifying course designation 为准。
- CMSC 798 / CMSC 799 是 research / thesis 的学分入口；是否计入 thesis 或 non-thesis option 需按 MS policy 和 advisor approval 执行。
