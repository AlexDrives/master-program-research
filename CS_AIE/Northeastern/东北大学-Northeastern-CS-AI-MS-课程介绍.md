# 东北大学 Northeastern - Computer Science / Artificial Intelligence MS 课程介绍

> 来源：Northeastern Graduate Catalog、Khoury College、course descriptions 官方页面  
> 核验日期：2026-07-06  
> 覆盖项目：M.S. in Computer Science (MSCS, Boston)、M.S. in Artificial Intelligence (Boston)

---

## 项目概况

- **MS Computer Science (MSCS)**：32 semester hours，包含 programming / algorithms core、breadth areas 和 electives。Breadth 需从三个 breadth areas 中的两个区域完成三门课。
- **MS Artificial Intelligence (Boston)**：32 semester hours，跨 Khoury College of Computer Sciences 与 College of Engineering；学生按 concentration 入学并由对应学院授予学位。官方 Boston 版 concentration 包括 Computer Vision、Continuous Process Engineering、Energy Systems、Machine Learning、Robotics and Agent-Based Systems。
- 本文件完整列出 catalog 中给出的 core、breadth、electives、concentration course pools；机器人 / 感知相关课程只在完整结构后单独标注。

## 项目一：M.S. in Computer Science (MSCS)

### 修读要求

| 项目 | 官方要求 |
|---|---|
| 总学分 | 32 semester hours |
| GPA | minimum 3.000 GPA |
| Core | Programming Design Paradigm and Recitation；Algorithms |
| Breadth | Complete three courses from two breadth areas，共 12 hours |
| Electives | 12 semester hours |
| Thesis / Project | CS 7990 Thesis、CS 8674 Master's Project 可作为 electives |

### Core Requirements

| 代码 | 课程名 | Hours |
|---|---|---:|
| CS 5010 / CS 5011 | Programming Design Paradigm and Recitation | 4 |
| CS 5800 | Algorithms | 4 |

### Breadth Areas - 完整官方课程池

| Breadth area | 官方课程池 |
|---|---|
| Artificial Intelligence and Data Science | CS 5100 Foundations of Artificial Intelligence；CS 5150 Game Artificial Intelligence；CS 5200 Database Management Systems；CS 5330 Pattern Recognition and Computer Vision；CS 6120 Natural Language Processing；CS 6140 Machine Learning；CS 6200 Information Retrieval；CS 6220 Data Mining Techniques；CS 6240 Large-Scale Parallel Data Processing；CS 7140 Advanced Machine Learning |
| Systems and Software | CS 5400 Principles of Programming Language；CS 5500 Foundations of Software Engineering；CS 5520 Mobile Application Development；CS 5600 Computer Systems；CS 5610 Web Development；CS 5700 Fundamentals of Computer Networking；CS 5850 Building Game Engines；CS 6410 Compilers；CS 6510 Advanced Software Development；CS 6620 Fundamentals of Cloud Computing；CS 6650 Building Scalable Distributed Systems |
| Theory and Security | CS 6760 Privacy, Security, and Usability；CS 7805 Complexity Theory；CY 5770 Software Vulnerabilities and Security；CY 6740 Network Security |

### Electives - 完整官方课程池

Complete 12 semester hours from the following:

- CS 5097 Mixed Reality

- CS 8982 Readings

## 项目二：M.S. in Artificial Intelligence (Boston)

### 修读要求

| 项目 | 官方要求 |
|---|---|
| 总学分 | 32 semester hours |
| GPA | minimum 3.000 GPA |
| Core | 16 hours；core courses 需 cumulative GPA 3.000 or higher |
| Concentration | 完成一个 concentration |
| 学院归属 | 学生录入其 concentration 所属学院，并遵循该学院政策；degree 由该学院授予 |

### Core Requirements

| 代码 | 课程名 | Hours |
|---|---|---:|
| CS 5100 | Foundations of Artificial Intelligence | 4 |
| CS 5130 | Applied Programming and Data Processing for AI | 4 |
| DADS 5200 或 DS 5020 | Mathematics for Machine Learning / Introduction to Linear Algebra and Probability for Data Science | 4 |
| EECE 5644 或 DADS 7275 | Introduction to Machine Learning and Pattern Recognition / Machine Learning and Data Analytics | 4 |

### Concentration 1：Computer Vision - College of Engineering

| 模块 | 官方课程池 | Hours |
|---|---|---:|
| Complete 8 hours | EECE 5550 Mobile Robotics；EECE 5554 Robotics Sensing and Navigation 或 ME 5554 Robotics Sensing and Navigation；EECE 5614 Reinforcement Learning and Decision Making Under Uncertainty；EECE 5639 Computer Vision；EECE 5642 Data Visualization | 8 |

## 官方课程描述抓手

- MSCS：AI/Data Science breadth 中 CS 5330、CS 6140、CS 7140 等适合感知 / 机器学习；但 Systems and Software、Theory and Security 同样是官方 breadth pool，不能删掉。
- MSAI：Computer Vision concentration 和 Robotics and Agent-Based Systems concentration 与机器人 / 感知最直接；Machine Learning concentration 提供 RL、CV、NLP、Deep Learning 等基础。
- 课程描述可在 Northeastern course descriptions 中检索 CS、EECE、CHME、ENSY、ME、PHIL、PHSC、CY 等课程代码。
