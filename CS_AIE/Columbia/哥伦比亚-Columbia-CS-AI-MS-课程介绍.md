# 哥伦比亚大学 Columbia - Computer Science / Artificial Intelligence MS 课程介绍

> 来源：Columbia Computer Science、Columbia Engineering、Columbia AI MS 官方项目 / curriculum / track / bulletin 页面  
> 核验日期：2026-07-06  
> 覆盖项目：M.S. in Computer Science、M.S. in Artificial Intelligence

---

## 项目概况

- **M.S. in Computer Science**：30-point CS 硕士，学生按 track 组织选课。官方 CS MS 页面要求课程为 4000-level 及以上，并设置 breadth、track、6000-level technical courses、GPA、PDL 等要求。
- **M.S. in Artificial Intelligence**：30-point、full-time、STEM-designated 项目，官方结构为 AI Core、Concentration、Electives / Capstone；可选择 Robotics and Perception 等 concentration。
- 本文件先完整记录官方给出的项目结构、track / concentration 框架和公开课程池，再在后面单独标注机器人 / 感知相关课程；不因课程不属于机器人方向而删除。

## 项目一：M.S. in Computer Science

### 修读要求

| 项目 | 官方要求 |
|---|---|
| 总学分 | 30 points |
| 课程层级 | 4000-level 及以上 |
| GPA | 至少 2.7 |
| Breadth | 需完成 CS breadth requirement |
| 6000-level | 至少 6 points technical courses at 6000 level |
| PDL | Professional Development and Leadership requirement |
| Track | 学生选择一个 track，并按 track 页面完成 required / elective course requirements |

### 官方 track / 课程结构

Columbia CS MS 官方页面列出多个 track。下表保留完整 track 框架；各 track 的年度课程可用性、课程编号变化和 adviser 批准规则以对应 track 页面、Directory of Classes、Bulletin 为准。

| Track | 官方方向说明 / 课程池抓手 |
|---|---|
| Machine Learning | 机器学习、统计学习、深度学习、概率建模、优化、AI applications；常见课程抓手包括 COMS W4771 Machine Learning、COMS W4721 Machine Learning for Data Science、COMS W4995 / E6998 topics、COMS E6253 Advanced Machine Learning 等 |
| Natural Language Processing | NLP、speech / language、information extraction、dialogue、semantic analysis；常见课程抓手包括 COMS W4705 Natural Language Processing、COMS W4706 Spoken Language Processing、COMS E6998 / W4995 NLP topics 等 |
| Vision, Graphics, Interaction, and Robotics (VGIR) | vision、graphics、HCI、robotics；官方 track 页面列出 Computer Vision、Computer Vision II、Computational Aspects of Robotics、Machine Learning、Advanced Machine Learning and Perception、Humanoid Robots 等课程抓手 |
| Network Systems | networks、distributed systems、internet / cloud systems、mobile / wireless systems；课程抓手来自 track 页面与 CS course catalog |
| Software Systems | programming languages、software engineering、databases、operating systems、distributed systems、compilers 等系统软件课程 |
| Foundations of Computer Science | algorithms、complexity、cryptography、logic、theory 等理论课程 |
| Computer Security | security、privacy、cryptography、systems security、network security 等安全课程；2026-07-06 直接访问旧 track URL 返回 404，需从当前 CS MS 页面或 Bulletin 进入最新版链接复核 |
| Computational Biology | algorithms / ML / statistics for biological data、computational genomics、systems biology 等交叉课程 |

## 项目二：M.S. in Artificial Intelligence

### 修读要求

| 项目 | 官方要求 |
|---|---|
| 总学分 | 30 points |
| 项目类型 | Full-time, STEM-designated |
| AI Core | 12 points |
| Concentration | 12 points |
| Electives / Capstone | 6 points electives 或 interdisciplinary capstone |
| Research substitution | Faculty-supervised AI research 可替代 6 credits electives / capstone |
| PDL | ENGI E4000 Professional Development and Leadership required |

### 官方课程结构 / 全部课程池

| 模块 | Points | 官方要求 / 课程池 |
|---|---:|---|
| AI Core | 12 | 从 AI / ML / NLP / Computer Vision 等核心 AI 课程中修读；课程池以 MS AI Curriculum 页面当年列表为准 |
| Concentration | 12 | 官方 concentration 包括 Robotics and Perception 等方向；每个 concentration 使用其页面列出的课程池 |
| Electives / Capstone | 6 | 可选 electives 或 interdisciplinary capstone；faculty-supervised AI research 可替代该 6 credits |
| PDL | 0 | ENGI E4000 Professional Development and Leadership |

### Robotics and Perception concentration - 官方课程抓手

| 类别 | 课程 / 主题 |
|---|---|
| Vision | Computer Vision I；Computer Vision II；Computational Imaging；Computational Photography |
| Robotics | Computational Aspects of Robotics；Humanoid Robots；robotics topics |
| ML / Perception | Machine Learning；Advanced Machine Learning and Perception；Neural Networks and Deep Learning |
| AI systems | Agentic Systems；AIoT；Edge AI |

## 官方课程描述抓手

- CS MS 的课程编号、课程描述、开课学期以 Columbia CS track pages、Columbia Bulletin 和 Directory of Classes 为准。
- MS AI 的课程结构以 Columbia Engineering MS AI Curriculum 页面为准；具体 AI Core、Concentration、Elective / Capstone 当年课程池需在 curriculum 页面和 Bulletin 中核对。
- 机器人 / 感知重点抓手：COMS W4731、COMS W4732、COMS W4733、COMS W4771、COMS W4772、COMS E6731，以及 MS AI Robotics and Perception concentration 下的 vision / robotics / ML-perception / AI systems 课程。
