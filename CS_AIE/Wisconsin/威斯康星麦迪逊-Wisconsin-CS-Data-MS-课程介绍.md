# 威斯康星大学麦迪逊分校 - CS / Data MS 课程介绍

> 来源：UW-Madison Computer Sciences、Statistics 官方项目页与 Graduate Guide；2026-07-13 核验。  
> 包含项目：`Traditional Computer Sciences MS`、`Computer Sciences: Professional Program MS (PMP)`、`MS Data Engineering (MSDE)`、`MS Data Science (MSDS)`。

---

## 项目概况与路径判断

| 项目 | 定位 | 机器人 / 感知判断 |
|---|---|---|
| Traditional CS MS | 研究型的传统 CS MS/PhD 体系中的 MS named option；可做 project 或 thesis。 | **可通过选课、导师与 thesis 做相关研究，但没有 Robotics/Perception 专项或固定课程路径。** |
| Professional CS MS (PMP) | 面向希望获得更专门 CS 教育的 professional MS；官方称可在两年内完成。 | **不是机器人项目。** 可修 AI、graphics、HCI 等广泛 CS 课，但官方没有机器人/感知必修轨。 |
| MS Data Engineering | 面向大规模数据管理的专业型 MS。 | **不是机器人/感知路径。** 核心是 distributed systems、big data、database、data integration。 |
| MS Data Science | Statistics 与 Computer Sciences 的跨学科数据科学 MS。 | **不是机器人项目。** 课程池含 computer vision / ML，但学位主线是统计、数据科学与数据系统。 |

四个项目均须满足 Graduate School 基线：至少 `30 credits`、至少 `16` residence credits、至少 `15` graduate-level credits、overall graduate GPA `3.00`；下列为各项目另行规定的结构。课程名称与是否当期开设均以官方 catalog / Course Search & Enroll 为准。

## 1. Traditional Computer Sciences MS

### 修读要求

| 模块 | 官方要求 |
|---|---|
| 学位总量 | `30 credits`；至少 `16` residence credits、`15` graduate-level credits，GPA 至少 `3.00`。 |
| CS 主体 | 至少 `24 credits` 为 `COMP SCI 400+`。 |
| Core credits | 上述 24 credits 中至少 `15 credits` 为按 A-F 计分的 `COMP SCI 700-889`。 |
| 24-credit 排除项 | `COMP SCI 400`、`COMP SCI/BME/BMI/BIOCHEM/CBE/GENETICS 915`、individual instruction `699/799/899`、`COMP SCI 702` 不可计入该 24 credits。 |
| Thesis / project | `COMP SCI 790`：department-approved project report 最多 `3 credits`；提交 departmental technical report 且由合规 thesis committee 批准的 thesis 最多 `6 credits`。通常可算 core，最终是否算 core 由 thesis supervisor/committee 在评定时决定。 |
| Topics 限制 | `COMP SCI 758`、`839`、`880` 三类 topics 合计最多一门可作 core；`COMP SCI 838` 不可作 core。 |
| 其余学分 | 剩余 `6 credits` 可来自任意 subject。`915` 可多次修读但不代替上述 CS 主体要求。 |

### 官方课程池 / 结构

- **不是固定的逐门 whitelist**：Guide 将 degree pool 定义为 `COMP SCI 400+`，其中 core 为 A-F 的 `COMP SCI 700-889`，再以排除项和 topics 上限约束；因此不能把该项目误写成只允许 AI/视觉课。
- 学位内明确点名的课程：`COMP SCI 702 Graduate Cooperative Education`、`758`、`790 Master's Thesis`、`799 Master's Research`、`838`、`839`、`880`、`899`，以及 `COMP SCI/BME/BMI/BIOCHEM/CBE/GENETICS 915 Computation and Informatics in Biology and Medicine`。
- 机器人/感知相关性只取决于在上述开放 CS 课程池中的实际选课、faculty advising 和 thesis 题目；官方学位要求本身**未列 Robotics 或 Perception concentration**。

## 2. Professional CS MS (PMP)

### 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | `30 credits`；至少 `16` residence、`15` graduate-level，GPA `3.00`。 |
| Core graduate CS | 至少 `15 credits` 为 `COMP SCI 700-889`。修 `838`、`839`、`880` 前须由 Professional Program Committee 确认可否满足项目 credit 要求。 |
| 其余学分 | 剩余学分为 `COMP SCI 400+`。 |
| Research / thesis / internship | `COMP SCI 799 Master's Research`、`790 Master's Thesis`（选择 thesis 时）与 `702 Graduate Cooperative Education` 合计最多 `6 credits`，且只能计入上述 15 core credits；超过 6 credits 不计入学位。`702` 最多修两次。 |
| Seminar / 不计入课程 | `COMP SCI 900 Advanced Seminar` 最多 `3 credits`，计入剩余 `400+` 部分；`915` 与 `699` 不计入项目要求。 |
| 校外课程 | 无 faculty advisor 与 program director 批准，不得修 prescribed curriculum 之外课程；不可与其他本科/研究生学位并行注册。 |
| 转学分 | 不接受其他机构的 graduate transfer credit；经批准最多转入 UW-Madison 本科的 `7 credits`，或 University Special student 的 `14 credits`（均受官方条件限制）。 |

### 官方课程池 / 结构

- **不是固定的 AI/机器人课表**：核心池为 `COMP SCI 700-889`，其余为 `COMP SCI 400+`；项目页仅列可涉猎领域：AI、computational biology、architecture、graphics、networks、security、databases、HCI、numerical analysis、optimization、performance、programming languages/compilers、systems、theory。
- 明确列出的结构性课程为 `COMP SCI 702`、`790`、`799`、`838`、`839`、`880`、`900`、`915`、`699`；其中 `839` 可重复修读但不可重修同一 topic。
- 因而该项目虽可从 AI、graphics 或 HCI 课程中形成相邻背景，**官方没有机器人、自动驾驶、视觉感知的 named option、必修 sequence 或 lab matching 承诺**。

## 3. MS Data Engineering (MSDE)

### 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | `30 credits`；至少 `16` residence、`15` graduate-level，GPA `3.00`。 |
| Data Engineering Foundations | 必修 `COMP SCI 739`、`744`、`764`、`774`，共 `12 credits`。 |
| ML breadth | 下列 ML 池至少选 `2` 门，共至少 `6 credits`。 |
| Algorithms breadth | 下列 algorithms 池至少选 `1` 门，`3-4 credits`。 |
| Systems breadth | 下列 systems 池至少选 `1` 门，`3-4 credits`。 |
| Humans and Data | 下列池至少选 `1` 门，`3 credits`。 |
| Electives | 以上课程或官方 elective pool 补足至 `30 credits`；同一门不得同时满足 foundation/breadth 与 elective。 |
| Thesis / project / internship | 可选 `CS 790` thesis（6 credits）或 project report（3 credits）；`799`、`790`、`702`、`900` 有可计入学位的总量限制，按 Guide 当年脚注执行。项目页说明 `CS 702` 可用于与学位直接相关的 internship。 |
| 范围限制 | 未经 Professional Programs Committee 批准，不得修 MSDE curriculum 外课程。 |

### 官方课程池（完整 Guide 表）

| 模块 | 官方课程池 |
|---|---|
| Foundations，全部必修 | `COMP SCI 739 Distributed Systems`; `744 Big Data Systems`; `764 Topics in Database Management Systems`; `774 Data Exploration, Cleaning, and Integration for Data Science` |
| Machine Learning，至少 2 | `COMP SCI 540 Introduction to Artificial Intelligence`; `COMP SCI/ECE 760 Machine Learning`; `COMP SCI 762 Advanced Deep Learning`; `COMP SCI/ECE 763 Trustworthy Artificial Intelligence`; `STAT 451 Introduction to Machine Learning and Statistical Pattern Classification`; `STAT 453 Introduction to Deep Learning and Generative Models`; `STAT 615 Statistical Learning` |
| Algorithms，至少 1 | `COMP SCI/ECE/ISYE 524 Introduction to Optimization`; `COMP SCI 577 Introduction to Algorithms`; `COMP SCI/ISYE/MATH/STAT 726 Nonlinear Optimization I`; `COMP SCI 787 Advanced Algorithms` |
| Systems，至少 1 | `COMP SCI 407 Foundations of Mobile Systems and Applications`; `537 Introduction to Operating Systems`; `564 Database Management Systems: Design and Implementation`; `640 Introduction to Computer Networks`; `COMP SCI/ECE 707 Mobile and Wireless Networking`; `740 Advanced Computer Networks` |
| Humans and Data，至少 1 | `COMP SCI 570 Introduction to Human-Computer Interaction`; `571 Building User Interfaces`; `765 Data Visualization`; `COMP SCI/ED PSYCH/PSYCH 770 Human-Computer Interaction` |
| Electives | 上述所有课程，或 `COMP SCI 642 Introduction to Information Security`; `702 Graduate Cooperative Education`; `790 Master's Thesis`; `799 Master's Research`; `900 Advanced Seminar in Computer Science`; `STAT 611 Statistical Models for Data Science`; `612 Statistical Inference for Data Science`; `613 Statistical Methods for Data Science` |

### 路径说明

- 该项目的不可替代主线是 `739/744/764/774` 的分布式系统、big data、数据库与 data integration；ML 只占 breadth 的一部分。
- `540/760/762/763` 可支持 ML 能力，但页面没有 robotics、motion planning、SLAM、robot perception 或 computer vision 的 requirement。它是**数据平台 / 数据工程项目，不是机器人或感知 MS**。

## 4. MS Data Science (MSDS)

### 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分与进度 | `30 credits`；至少 `16` residence、`15` graduate-level，GPA `3.00`。通常应在 `3-4 semesters` 完成；part-time 须获 program chair 批准。 |
| Statistics core | `STAT 611`、`612`、`613`，共 `9 credits`。 |
| CS core | Algorithms、Systems、Humans and Data 三类各选 `1`，共 `9 credits`。 |
| ML core | 从官方 ML pool 选 `2`，共 `6 credits`。 |
| Electives | 从 Data Science Electives 表选 `6 credits`；核心课与 elective 重叠时不得双重计数。`COMP SCI 799` 与/或 `STAT 699` 合计最多 `3 credits`。 |
| 转学分 | 经批准，其他机构 graduate coursework 最多 `9 credits`；UW-Madison 本科最多 `7 credits`；均受 Guide 年限及 Graduate School 条件限制。 |

### 官方课程池（完整 Guide 表）

| 模块 | 官方课程池 |
|---|---|
| Statistics core，全部必修 | `STAT 611 Statistical Models for Data Science`; `612 Statistical Inference for Data Science`; `613 Statistical Methods for Data Science` |
| CS core - Algorithms，选 1 | `COMP SCI/ECE/ISYE 524 Introduction to Optimization`; `COMP SCI 577 Introduction to Algorithms`; `COMP SCI/ISYE/MATH/STAT 726 Nonlinear Optimization I` |
| CS core - Systems，选 1 | `COMP SCI 537 Introduction to Operating Systems`; `544 Introduction to Big Data Systems`; `564 Database Management Systems: Design and Implementation`; `640 Introduction to Computer Networks`; `642 Introduction to Information Security`; `739 Distributed Systems`; `744 Big Data Systems`; `764 Topics in Database Management Systems` |
| CS core - Humans and Data，选 1 | `COMP SCI 765 Data Visualization`; `COMP SCI/ED PSYCH/PSYCH 770 Human-Computer Interaction` |
| ML core，选 2 | `COMP SCI 540 Introduction to Artificial Intelligence`; `COMP SCI/ECE 760 Machine Learning`; `COMP SCI/ECE 761 Mathematical Foundations of Machine Learning`; `COMP SCI 762 Advanced Deep Learning`; `STAT 451 Introduction to Machine Learning and Statistical Pattern Classification`; `STAT 453 Introduction to Deep Learning and Generative Models`; `STAT 615 Statistical Learning` |
| Data Science electives，选 6 credits | `COMP SCI/ECE/ISYE 524`; `COMP SCI 537`; `541 Theory & Algorithms for Data Science`; `544`; `564`; `COMP SCI/BMI 576 Introduction to Bioinformatics`; `577`; `640`; `642`; `702 Graduate Cooperative Education`; `COMP SCI/ISYE/MATH/STAT 726`; `736 Advanced Operating Systems`; `739`; `744`; `COMP SCI/ECE 763 Trustworthy Artificial Intelligence`; `764`; `765`; `COMP SCI/ECE 766 Computer Vision`; `COMP SCI/BMI 767 Computational Methods for Medical Image Analysis`; `769 Advanced Natural Language Processing`; `COMP SCI/ED PSYCH/PSYCH 770`; `COMP SCI/BMI 771 Learning Based Methods for Computer Vision`; `774`; `784 Foundations of Data Management`; `799 Master's Research`; `COMP SCI/ECE/STAT 861 Theoretical Foundations of Machine Learning`; `LIS 461 Data and Algorithms: Ethics and Policy`; `STAT 303/304/305 R for Statistics I/II/III`; `STAT 349 Introduction to Time Series`; `351 Introductory Nonparametric Statistics`; `STAT/COMP SCI 403 Internship Course in Comp Sci and Data Science`; `STAT 411 An Introduction to Sample Survey Theory and Methods`; `421 Applied Categorical Data Analysis`; `433 Data Science with R`; `443 Classification and Regression Trees`; `456 Applied Multivariate Analysis`; `461 Financial Statistics`; `STAT/COMP SCI 471 Introduction to Computational Statistics`; `575 Statistical Methods for Spatial Data`; `STAT/BMI 620 Statistics in Human Genetics`; `699 Directed Study`; `701 Applied Time Series Analysis, Forecasting and Control I`; `760 Multivariate Analysis I`; `761 Decision Trees for Multivariate Analysis`; `771 Computational Statistics`; `STAT/ECON/GEN BUS 775 Bayesian Statistics`; `ISYE 620 Simulation Modeling and Analysis`; `ISYE 624 Stochastic Modeling Techniques`; `ISYE/COMP SCI 719 Stochastic Programming`; `ISYE/COMP SCI 723 Dynamic Programming and Associated Topics`; `ISYE/COMP SCI/MATH 728 Integer Optimization`; `MATH 616 Data-Driven Dynamical Systems, Stochastic Modeling and Prediction` |

### 路径说明

- 课程池确有 `COMP SCI/ECE 766 Computer Vision`、`COMP SCI/BMI 767`、`771` 等感知相邻课，也有深度学习与 ML；它们只在 **6-credit elective** 的宽池中出现，非必修机器人序列。
- 没有 robotics、control、planning、SLAM 或 embodied systems 的项目要求。故 MSDS 适合数据、统计、ML、可视化方向，**不应标为机器人/感知专门项目**。

## 官网核验（2026-07-13）

- 已核验 Wisconsin Graduate Guide 对 Traditional CS MS、Professional CS MS、MSDE、MSDS 的 `30 credits`、residence、graduate coursework、GPA 与各自课程结构。
- 已逐项保留 MSDE 和 MSDS Guide 表列出的 foundation、core、breadth 与 elective course pools；未将非 AI 课程删去。
- Traditional CS MS 与 Professional CS MS 的 Guide 不给固定逐门选课表，而按 `COMP SCI` 课程号范围、core 定义、排除项和学分上限规定；本文按该官方粒度记录，未虚构 concentration。
