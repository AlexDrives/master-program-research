# 卡内基梅隆大学 CMU - MS in Information Networking (MSIN) 课程介绍

> 来源：CMU Information Networking Institute (INI) 官方 MSIN 项目页、`Curriculum 2026-27 (MS38)` 与 `MSIN Degree Specific Core Course List`（2026-07-13 核验）。
> 项目：`Master of Science in Information Networking (MSIN)`。

---

## 项目概况

- MSIN 是 CMU Information Networking Institute 的旗舰硕士，交叉计算机科学、ECE、软件工程、信息系统和业务/管理课程。
- 项目重点是 networked software、distributed systems、networking、security 与 systems engineering；它**不是** Robotics 或 Computer Vision 专项硕士。
- Standard option 为 `3 semesters / 123 units`；Advanced option 为 `4 semesters / 159 units`，在标准课程以外完成 `36 units` research thesis、development-based project 或 area of concentration。
- 项目面向 2026-27 入学的 MS38 cohort；实际课程开设、先修和跨院选课名额以当学期 Schedule of Classes 与 adviser 为准。

## 修读要求

| 模块 | Standard option | Advanced option | 官方要求 |
|---|---:|---:|---|
| Core | 75 units | 75 units | Systems、Networking、Business & Management、Security、APD、degree-specific core |
| Program electives | 48 units | 48 units | INI、ECE、SCS、EPP、ETC、III，或获批 Heinz/Tepper 课程 |
| Advanced study | - | 36 units | research thesis、development-based project 或 area of concentration |
| 总计 | 123 units | 159 units | 3 或 4 个学期 |

### 额外先修

`14-513 Introduction to Computer Systems` 是 MSIN 学生必须完成的先修课，也是 `14-735`、`14-736`、`14-642`、`15-605`、`15-640` 等系统课的先修。官网强烈建议第一学期修读；是否可因既有课程背景满足/豁免，应向 INI adviser 确认。

## Program Electives：48 units

- 选修可来自 INI、ECE、SCS、Engineering & Public Policy、Entertainment Technology Center、Integrated Innovation Institute，以及获批 Heinz College/Tepper School 课程。
- 官网列出的选修例子包括 `Cloud Computing`、`Machine Learning`、`Ethical Penetration Testing`；MSIN 项目页另举 `Applied Information Assurance`、`Wireless Protocols and Standards for IoT`。
- 项目没有给出一张穷尽全部院系选修课的固定清单；因此不能把跨院可选范围误写成“保证可选”。课程开设、席位与 adviser approval 均以当期为准。
- 国际学生如将 internship 作为选修，需要满足 CPT 资格并额外注册 3-unit internship course；Fall 入学通常可在首个暑期申请 CPT，Spring 入学 Standard option 首个暑期通常不满足资格。

## 可执行选课方向

### Systems / networked AI 路线

| 模块 | 推荐抓手 | 与机器人感知的关系 |
|---|---|---|
| Systems Core | `14-642` + `14-736`，或 `15-605` + `15-640` | 嵌入式、分布式与系统实现底座 |
| Networking Core | `18-756` 或 `15-641` | 机器人群体、边缘/云协同与网络通信基础 |
| Degree-Specific Core | `15-642 Machine Learning Systems` 或 `14-744 Mobile and Embedded Software Design` | 更贴 on-device learning、部署和机器人软件系统 |
| Program Electives | ML、cloud、wireless/IoT、embedded、security 相关获批课程 | 可构建 embodied system 的工程侧能力 |

### 与你的方向的真实匹配度

- 机器狗 RL navigation：可从 embedded systems、distributed systems、networking、ML systems 和 wireless/IoT 切入，强调 on-device/connected autonomy 的部署与可靠性。
- Affordance / environment reasoning 视觉模型：MSIN 没有官方 CV/perception core；只能通过获批 SCS/ECE electives 补 ML systems 或 AI 系统能力。
- 因而 MSIN 是 `robotics software / edge systems / networked autonomy` 的补充项，不应和 CMU MSCV、MSR、MRSD 或明确 robotics/perception 项目等量齐观。

### 核验结论（2026-07-13）

- MS38 Standard option 为 `123 units = 75 core + 48 electives`；Advanced option 为 `159 units`，另加 `36 units` advanced study。
- 75-unit core 由 Systems 24、Networking 12、Business 12、Security 12、APD 3、degree-specific core 12 构成。
- `14-513` 是项目要求的系统先修，官网建议第一学期完成。
- 官网明确提示：跨院课程不保证某学期开设或获得席位；选课须以 Schedule of Classes 和 adviser 确认为准。
