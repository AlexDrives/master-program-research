# 纽约大学 Courant - Computer Science MS 课程介绍

> 来源：NYU Courant MSCS requirements 与 Graduate Course Catalog，2026-07-06 核验  
> 包含项目：`M.S. in Computer Science`

---

## 项目概况

- NYU Courant MSCS 标准路径为 `30 credits`，也可选择带 concentration 的 `36 credits` 路径。
- 项目要求由 approved coursework、foundational courses、application area、capstone/thesis/advanced lab 构成。
- Courant MSCS 不是机器人专门项目；机器人/感知相关内容主要落在 AI、ML、CV、graphics、systems 与 project/thesis 中。

## 修读要求

| 项目 | 30-credit option | 36-credit option with concentration |
|---|---|---|
| 总学分 | 30 credits | 36 credits |
| GPA | cumulative GPA `3.0` | cumulative GPA `3.0` |
| Standard CS classroom courses | 至少 21 credits | 至少 21 credits |
| Additional credits | 6 credits 可为 standard graduate CS / Math / Data Science classroom courses、independent study 或 MS thesis；independent study 和 thesis 需 DGS approval | 同左 |
| Remaining credits | 3 credits 可为上述类别、transfer credit、external internship 或经批准 NYU 其他院系 graduate courses；external internship 最多 3 credits | 9 credits 可为上述类别；external internship 最多 6 credits |
| Foundational courses | 3 门；三门 attempted foundational 平均需 B- / 2.667 或以上 | 同左 |
| Application / concentration | 通过 1 门 designated application area course | 完成 AI 或 Systems/Security concentration 的 4 门课 |
| Capstone | designated capstone，成绩 B 或以上；可在条件满足时用 thesis 或 advanced lab 替代 | concentration capstone 必修，成绩 B 或以上；thesis 只能作 elective credit，不能替代 capstone |

## Thesis / Capstone 规则

| 项目 | 官方规则 |
|---|---|
| Capstone | DGS 指定；需 substantial programming effort；成绩 B 或以上 |
| Capstone examples | Advanced Computer Graphics、Advanced Database Systems、Compiler Construction |
| Thesis 替代 capstone | 30-credit option 可在 DGS approval 下选择 master's thesis |
| Thesis 条件 | 6 门课后 cumulative GPA `3.75`；3 门 foundational 每门至少 `B+`；找到 full-time faculty advisor；DGS approval |
| Thesis 学分 | 6 credits |
| Thesis 产出 | 原创 research 或 design/implementation；工作量等同两门 MS courses；30-50+ pages；两名 full-time faculty 批准 |
| 36-credit concentration | thesis optional，但只计 elective credit，不能替代 concentration capstone |

## 官方课程描述抓手

- 感知 / 视觉 / 图形：`CSCI-GA 2270`, `2271`, `2274`, `2585`。
- NLP / language intelligence：`CSCI-GA 2590`, `2591`。
- 系统支撑：`CSCI-GA 2250`, `2262`, `2620`, `2621`, `3250`。
- 官网 catalog 本轮未见 robotics-titled Courant GA 课程；不应把 MSCS 写成机器人项目。
