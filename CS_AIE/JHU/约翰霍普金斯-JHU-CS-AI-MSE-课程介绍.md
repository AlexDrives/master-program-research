# 约翰霍普金斯大学 - Computer Science MSE / AI Track 课程介绍

> 官网核验日期：2026-07-06  
> 项目：M.S.E. in Computer Science；AI Master's Track

---

## 项目概况

- JHU CS MSE 可走 general MSE，也可选择 AI master's track。
- General MSE 要求 approved courses 并覆盖 core areas；AI track 是 CS master 内的 AI 路线。
- AI track 明确给出 core areas 与完整 AI course list，覆盖 CS、ECE、AMS、Cognitive Science、Philosophy、Psychology and Brain Sciences。

## 项目一：M.S.E. in Computer Science

### 修读要求

| 模块 | 官方要求 |
|---|---|
| Course requirement | 8 one-semester courses in CS 或 closely related fields，经 faculty advisor 批准 |
| Core areas | 至少 4 个 core areas 中各 1 门 |
| Core area list | Application, Reasoning, Software, Systems, Theory |
| Research option | 可用 faculty-guided research project 组合完成 |
| Coursework option | 可通过 10 approved courses 完成 |
| Track / concentration | 可不选 track；HLT 是 official concentration；AI track 为 broader AI route |

### 官方 core area 结构

| Core area | 说明 |
|---|---|
| Application | 应用方向 |
| Reasoning | 推理、AI、理论化 reasoning |
| Software | 软件、编程语言、软件工程 |
| Systems | 系统、网络、架构 |
| Theory | 理论、算法 |

## 项目二：AI Master's Track

### 修读要求

| 路线 | 官方要求 |
|---|---|
| Coursework route | 5 AI courses |
| Project/thesis route | 4 AI courses + AI master's project/thesis |
| Core coverage | Machine Learning、Text and Speech、Vision and Robotics 三个 core areas 各至少 1 门 |

### AI track core areas

| Core area | 课程 |
|---|---|
| Machine Learning | 601.675 Machine Learning；601.682 ML: Deep Learning；601.674 Machine Learning Theory；601.684 Explainable AI Design & Human-AI Interaction |
| Text and Speech | 601.665 Natural Language Processing；601.667 Introduction to Human Language Technology；601.668 Machine Translation；601.671 NLP: Self-Supervised Models |
| Vision and Robotics | 601.661 Computer Vision；601.663 Algorithms for Sensor-Based Robotics；530.646 Robot Devices, Kinematics, Dynamics, and Control；601.685 Probabilistic Models of the Visual Cortex |
