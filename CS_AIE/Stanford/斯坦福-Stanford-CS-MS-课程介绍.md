# 斯坦福大学 - Computer Science MS 课程介绍

> 官网核验日期：2026-07-06  
> 项目：M.S. in Computer Science

---

## 项目概况

- Stanford CS MS 是以 approved academic plan / program sheet 管理的 45-unit 硕士项目。
- 学生需在 MS CS plan 激活后 3 年或 9 个非夏季 quarter 内完成要求。
- 官方预批准 specialization 包括：Artificial Intelligence, Computational Biology, Computer and Network Security, Human-Computer Interaction, Information Management and Analytics, Software Theory, Systems, Theoretical Computer Science, Visual Computing。
- 本文件保留完整官方结构；AI / robotics / perception 相关内容仅在后文单独标注，不替代完整课程结构。

## 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | 至少 45 units，且必须是在 Stanford 完成；外校课程不能转入抵 45 units |
| Program sheet | 每名学生必须提交并获批 program sheet；可在仍 active 的学年版本中选择一个版本，不可混用不同版本要求 |
| 时间限制 | 3 年或 9 个非夏季 quarter |
| 注册要求 | 秋、冬、春通常每 quarter 至少 8 units；HCP 和毕业前最后一学期有例外 |
| Graded units | Stanford Bulletin 显示至少 36 graded units，GPA 至少 3.0 |
| Foundations | 通过 Stanford CS 官方 Foundations requirement |
| Significant Implementation | 通过一门带 SI designation 的课程，可与 breadth / depth / elective 重叠 |
| Breadth | 在 4 个 breadth areas 中完成 3 门、覆盖 3 个不同 area，每门至少 3 units、letter grade |
| Depth | 在所选 specialization 中完成 21 units，每门至少 3 units、letter grade；CS 399 在 depth 中最多 6 units 且需批准 |
| Electives | 用经批准课程补足 45 units；需满足总体 graded units 和 program sheet 规则 |

## 官方课程结构 / 全部课程池

### 1. Breadth areas

Stanford MS CS 官方 breadth areas 共 4 个，要求从中选 3 个不同 area：

| Breadth area | 说明 |
|---|---|
| Formal Foundations | 形式化、理论、算法与数学基础 |
| Learning and Modeling | 学习、建模、AI / ML 相关基础 |
| Systems | 计算机系统、网络、架构、软件系统 |
| People and Society | HCI、社会影响、政策、伦理等人与社会方向 |

### 2. Approved specializations

| Specialization | 官方定位 / 课程池入口 |
|---|---|
| Artificial Intelligence | AI principles, logic, probability, language；主题包括 knowledge representation, reasoning, robotics, ML, probabilistic modeling, NLP, cognition, applications |
| Computational Biology | 生物与医学信息学中的计算问题；算法、数据库、网络、建模仿真、生物计算等 |
| Computer and Network Security | 计算机安全、网络安全、密码、系统安全 |
| Human-Computer Interaction | 人机交互、设计、交互系统、用户研究 |
| Information Management and Analytics | 数据管理、信息检索、数据挖掘、分析 |
| Software Theory | 软件、编程语言、形式方法、理论支撑 |
| Systems | 操作系统、网络、分布式系统、体系结构等 |
| Theoretical Computer Science | 算法、复杂性、理论计算机科学 |
| Visual Computing | 视觉计算、图形学、计算摄影、视觉与 AI / systems 的交叉 |

### 3. Public page 明确给出的 AI sample classes

| 课程 | 英文名称 |
|---|---|
| CS 221 | AI Principles & Techniques |
| CS 224N | Natural Language Processing with Deep Learning |
| CS 229 | Machine Learning |

### 4. 课程清单抓手

- 官方 MS Program Sheets 页面说明：每个 specialization 有独立 program sheet，当前学生通过 GIN 提交；Program Sheets / Stanford Bulletin 是 depth course 的正式清单入口。
- Stanford Bulletin 的 CS-MS 页面是公开 catalog 入口；具体课程描述可通过 Stanford Bulletin CS courses 和 ExploreCourses 查询。
- 本次核验中，公开页面能确认完整 degree structure 和全部 specialization 名称；逐个 specialization 的完整 PDF/program sheet 课程列表需要从 GIN 或当前可下载 program sheets 再人工复核。

## 官方课程描述抓手

- Stanford Bulletin CS courses 和 ExploreCourses 是逐门课程描述、学分、开课学期和 prerequisite 的公开抓手。
- 各 specialization 的 program sheet 是 depth / elective 课程归属的正式抓手；需按当前学年版本复核，不能混用不同年份表格。

## AI / 机器人 / 感知相关课程标注

- 最相关 specialization：Artificial Intelligence, Visual Computing。
- 相关主题：CS 221 AI, CS 229 ML, CS 224N NLP, robotics, probabilistic modeling, computer vision, graphics, visual computing。
- 注意：Stanford MS CS 不是 robotics 专门硕士；机器人/感知应作为 specialization 与 advisor/course plan 的组合来落地。
