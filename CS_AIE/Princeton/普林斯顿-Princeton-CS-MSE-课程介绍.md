# 普林斯顿大学 - Computer Science M.S.E. 课程介绍

> 来源：Princeton Computer Science 官方硕士学位与 M.S.E. Track 页面、官方 Course Schedule；2026-07-13 核验。  
> 项目：`Master of Science in Engineering (M.S.E.) in Computer Science`。

---

## 项目概况

- CS 系同时授予 `M.S.E.` 与 `M.Eng.` 两种全额资助的硕士学位；录取生起始进入要求论文的 M.S.E. track，第一学年春季 readmission 时可选择转入非论文 M.Eng. track。
- M.S.E. 是两年、全日制的课程加原创论文项目；所有课程必须以 letter grade 修读，且总修读时间不得超过两年。
- 本文件只说明 M.S.E. track；M.Eng. 的非论文要求不混入本文件。

## 修读要求

| 项目 | 官方要求 |
|---|---|
| 课程总量 | `6` 门：`3` 门 core + `3` 门 elective |
| 成绩 | 维持 `B average`，最多一门 `C`；所有计入课程必须取得 A-F letter grade |
| 第一年进度 | 至少修完 `4` 门；因排课无法完成时须获 Director of Graduate Studies 批准 |
| 第二年续读 | 第一学年结束前必须确认 thesis adviser，并提交经 adviser 批准的 `350-500` 词 thesis proposal abstract |
| 论文 | 原创 thesis；由 adviser 和另一名 reader 审阅、评分；须作公开研究报告 |
| 公开报告 | 第二年春季 `20` 分钟报告，随后 `10` 分钟问答；报告不计分 |
| 本科抵扣 | 仅 Princeton 本科生可按专项政策，最多以 `2` 门超出本科 COS 要求的高阶 COS 课计入；须满足硕士要求 |

## 官方课程结构 / 全部课程池

### Core Courses（任选 3 门）

官方规定三门 core 均须来自以下 COS 课程编号池；课程名称、先修与当学期是否开设以 Registrar / Course Schedule 为准。

`COS 475 (ELE 475)`；`COS 510`；`COS 511`；`COS 513`；`COS 514`；`COS 516`；`COS 518`；`COS 521`；`COS 522`；`COS 524`；`COS 525`；`COS 526`；`COS 527`；`COS 528`；`COS 529`；`COS 533`；`COS 534`；`COS 551 (MOL 551)`；`COS 557 (MOL 557)`；`COS 561`；`COS 563`；`COS 568`；`COS 583`；`COS 584`；`COS 585`。

### Elective Courses（任选 3 门）

每门 elective 须满足下列任一官方条件：

| 类别 | 官方课程池 / 规则 |
|---|---|
| 指定 COS 课 | `COS 318, 320, 324, 326, 343, 352, 375, 597, 598` |
| 高阶 COS 课 | 任意 `400-level COS` course |
| 补充 core | 另一门符合上述 core 课程池的课程 |
| 非 COS 选修 | 官方 non-COS elective list 中的课程；未列但相关的 non-COS 课程，可与 academic adviser 讨论并由 Director of Graduate Studies 批准后加入 |

### 论文与课程的关系

- 选课应服务于一个 CS 子领域的聚焦；论文在 adviser 指导下完成，不能以课程替代。
- 官方页面没有把六门课划分为机器人、视觉或 AI concentration；这些是学生在 core/elective 规则内自行组合的研究与选课主题。

## 机器人 / 视觉相关性

- 官方 `Fall 2026` Course Schedule 中，`COS 529 Advanced Computer Vision` 位于 M.S.E. core 课程池，是最直接的、可计入 core 的视觉课程。
- 同一官方课表列出 `COS 429 Computer Vision`（`400-level COS`，可按 elective 规则计入）与 `COS 345 Introduction to Robotics`（不在当前 M.S.E. 指定 elective 列表；是否可计入需按 non-COS/个案批准规则向项目确认）。
- 课表还列出 `COS 572 Safety-Critical Robotics and AI`，但它不在 M.S.E. 页面公布的 core 或指定 elective 编号池中；因此只能作为研究/开课信号，**不可据此假定可计入学位**。
- 对机器人感知最稳妥的已公开组合是 `COS 529`（core）加 `COS 429`（elective）；机器人控制、具身 AI 与跨系课程须在选课前取得项目确认。
