# 加州大学尔湾分校 - Master of Computer Science 课程介绍

> 官网核验日期：2026-07-06  
> 项目：Master of Computer Science (MCS)

---

## 项目概况

- UCI MCS 是面向 advanced professional practice 的 professional master。
- 项目通过 core subjects、recommended elective pathways 和 capstone / practicum 组织。
- 官方课程页明确列出 core curriculum、recommended elective pathways 和 sample timetable。

## 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | Catalogue 结构为 44 units |
| Plan | Plan II |
| Core subjects | 完成官方 core curriculum |
| Electives | 按 recommended pathways 或个人目标选择 electives |
| Experiential learning | Keystone / capstone / practicum sequence |
| Capstone | written/oral capstone presentation / project demonstration |
| 时长 | 官网定位为职业型、通常 15-16 months |

## 官方课程结构 / 全部课程池

### 1. Core curriculum

| 课程 | 英文名称 |
|---|---|
| COMPSCI 253P | Advanced Programming and Problem Solving |
| COMPSCI 260P | Algorithms with Applications |
| COMPSCI 261P | Data Structures with Applications |
| COMPSCI 295P | Keystone Project for Computer Science |
| COMPSCI 296P | Capstone Professional Writing and Communication for Computer Science Careers |
| COMPSCI 297P | Capstone Design Project for Computer Science |
| COMPSCI 298P | Computer Science Practicum |

### 2. Recommended elective pathways

| Pathway | 官方主题 |
|---|---|
| Intelligent Systems & AI | artificial intelligence, machine learning, deep learning, search, language technologies, intelligent information systems |
| Data Engineering & Analytics | databases, large-scale data systems, distributed data management, analytics, data-intensive application design |
| Computing Systems & Infrastructure | systems-level software, computer architecture, distributed computing, performance, scalable computing platforms |
| Additional elective subjects: Security | computer security and network security across multiple domains |
