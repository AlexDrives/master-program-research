# 密歇根大学安娜堡分校 - CSE MS/MSE 课程介绍

> 官网核验日期：2026-07-06  
> 项目：M.S. / M.S.E. in Computer Science and Engineering

---

## 项目概况

- UMich CSE master 主要面向希望系统提升 CSE 专业能力的学生。
- MS 与 MSE 的 degree requirements 相同，名称主要取决于学生是否有 engineering bachelor background。
- 项目要求 30 credits coursework；thesis optional；通常 1.5-2 年完成。
- 官方主页提示完整要求在 CSE Graduate Program Guide，课程要求列表在官方 Google Sheet；本文件先保留公开官网可核验结构。

## 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | 30 credits of coursework |
| 学位名称 | MS / MSE 要求相同；engineering bachelor 可选 MSE，非 engineering bachelor 通常为 MS |
| Thesis | Optional |
| 项目时长 | 通常 1.5-2 years |
| Transfer / equivalency | 可有限转入 graduate credits；等效课程可满足要求但不减少 30 credits 总量 |
| 课程规则 | 以 CSE Graduate Program Guide 和 Course Requirements Google Sheet 为准 |

### 2. CSE 官方研究 / emphasis 方向池

UMich CSE 官网与课程生态覆盖以下方向；它们不是固定 track requirement，而是选课和研究计划的方向池：

| Area | 说明 |
|---|---|
| Artificial Intelligence | AI, reasoning, planning |
| Machine Learning | ML, statistical learning, deep learning |
| Computer Vision | visual perception, image/video understanding |
| Robotics / Autonomous Systems | robotics, autonomous systems, navigation, interaction |
| Algorithms and Theory | algorithms, complexity, theory foundations |
| Architecture | computer architecture |
| Compilers and Programming Languages | compilers, PL |
| Databases and Data Mining | databases, data management, mining |
| Embedded and Mobile Systems | embedded/mobile computing |
| Human-Computer Interaction | HCI, interactive systems |
| Operating Systems, Networks, and Distributed Systems | systems, networks, distributed computing |
| Security and Privacy | security/privacy |
| Software Engineering | software systems and engineering |

## 官方课程描述抓手

- 逐门课程与“满足哪个 MS requirement”的归属需使用 UMich CSE 官方 Course Requirements Google Sheet。
- 课程描述、开课学期和编号需用 UMich course guide / bulletin 复核。
- 本次未把非公开或需 Google 权限动态加载的 sheet 内容转写为固定课程表，避免伪造课程池。

## AI / 机器人 / 感知相关课程标注

- 最相关方向：Artificial Intelligence, Machine Learning, Computer Vision, Robotics / Autonomous Systems, Embedded/Mobile Systems。
- 适合关注 robotics perception、autonomy、visual computing、learning-enabled systems 的学生，但具体课程可计入情况必须按当年 Course Requirements Sheet。
