# 加州大学洛杉矶分校 - CS MS / MEng Robotics & Autonomous Systems 课程介绍

> 来源：UCLA Graduate Program Requirements、UCLA MEng 官方页面，2026-07-06 核验  
> 包含项目：`M.S. in Computer Science`；`Master of Engineering - Robotics & Autonomous Systems`

---

## 项目概况

- UCLA Computer Science M.S. 是 Henry Samueli School of Engineering and Applied Science 下的 CS 硕士，可选择 Thesis Plan I 或 Capstone Plan II。
- UCLA MEng 是 one-year professional master，Robotics & Autonomous Systems 属于 technology concentration / direction，重视工程课程、professional development 与 `ENGR 299` capstone。
- 本文件完整记录官方给出的 degree requirements、breadth pools、fields / concentration 结构；机器人相关课程仅在后文单独标注。

## 项目一：M.S. in Computer Science

### 修读要求

| 项目 | Thesis Plan I | Capstone Plan II |
|---|---|---|
| 总课程 | 9 courses | 9 courses |
| Formal / CS courses | 7 formal courses，letter grade；其中至少 4 门为 200-level CS | 至少 5 门 200-level CS，letter grade |
| Thesis / Capstone | 2 courses / 8 units `CS 598` thesis work | individual capstone project，由 faculty adviser 指导，final report 由 3 名 faculty committee review |
| Electives | 其余 3 门可为 100/200-level CS 或 closely related discipline 的 200-level courses | 其余 4 门可为 100/200-level CS 或 closely related discipline 的 200-level courses |
| 不可计入 | `CS 201` seminars 不可计入 9 门课；500-level courses 不可用于 Plan II |
| Teaching / field experience | Not required |
| Time to degree | normative TTD 6 quarters；maximum TTD 7 quarters |
| GPA 风险线 | UCLA 页面列出 termination reason 包括所有课程与 200-series GPA 未维持 3.0、任意连续两学期 GPA 未维持 3.0 等 |

### 官方 Areas of Study

M.S. 学生不必选择 major field，可从以下 field 广泛组合课程：

| 官方 field |
|---|
| Artificial Intelligence |
| Computational Systems Biology |
| Computer Networks |
| Computer Science Theory |
| Computer System Architecture |
| Graphics and Vision |
| Data Science Computing |
| Software Systems |

### 官方 Breadth Requirement / 全部课程池

M.S. 学生必须在 UCLA graduate residence 的第 4 个 quarter 结束前满足 CS breadth requirement。官方要求掌握以下 5 组本科课程内容或等价内容：

| Breadth 组 | 官方课程池 |
|---|---|
| Algorithms | `Computer Science 180` |
| Systems / Architecture / Networks 组三选二 | `Computer Science 111`; `Computer Science 118`; `Computer Science M151B` |
| Databases / AI / Graphics 组三选一 | `Computer Science 143`; `Computer Science 161`; `Computer Science 174A` |
| Programming Languages / Software / Compiler 组三选一 | `Computer Science 130`; `Computer Science 131`; `Computer Science 132` |
| CS seminar | 至少 3 quarters of `Computer Science 201`，成绩 Satisfactory；但 `CS 201` 不计入 9 门 degree courses |

### Breadth Competence 的官方证明方式

| 方式 | 要求 |
|---|---|
| UCLA 修课 | 在 UCLA 完成对应课程，成绩 B- 或以上 |
| 外校等价课 | 在另一所大学完成等价课程，成绩 B- 或以上 |
| 考试 | 不注册该课，但通过 UCLA 课程通常使用的 midterm 与 final examination |

## 项目二：Master of Engineering - Robotics & Autonomous Systems

### 修读要求

| 项目 | 官方要求 |
|---|---|
| 项目定位 | One-year professional MEng |
| Concentration courses | 每个 technology concentration 通常要求 5 门 engineering courses |
| Professional development | 3 门 graduate-level professional development courses |
| Capstone | `ENGR 299`，industry-sponsored capstone project |
| Thesis | No thesis |
| 方向 | Robotics & Autonomous Systems / Autonomous Systems concentration |

### 官方课程结构 / 课程池抓手

UCLA MEng 页面将 Robotics & Autonomous Systems 描述为覆盖 autonomous systems、robotics、control、sensing、machine learning、optimization、embedded / cyber-physical systems 等工程主题。已核到的官方结构如下：

| 模块 | 官方结构 |
|---|---|
| Technology concentration | 5 门 engineering concentration courses |
| Professional development | 3 门 graduate-level professional development courses |
| Capstone | `ENGR 299` industry-sponsored capstone |
| Robotics / autonomy topics | control；kinematics / dynamics / sensing；machine learning；optimization；embedded and cyber-physical systems；autonomous systems |

> 说明：UCLA MEng 官方网页本轮可核到项目结构与 Robotics / Autonomous Systems 主题，但网页前端未稳定暴露可复制的逐门完整课程池。此处不自行补造课程清单；需要以 UCLA MEng 当年 concentration advising sheet 或 registrar 当季 catalog 人工复核逐门课程。

## 官方课程描述抓手

- UCLA CS MS 的机器人相关 field：`Artificial Intelligence`、`Graphics and Vision`、`Data Science Computing`、`Computer System Architecture`、`Software Systems`。
- CS breadth 中与 robot brain / perception 相关：`CS 161`（AI 类）、`CS 174A`（graphics 类）、`CS 180`（algorithms）、`CS 111/118/M151B`（systems / networks / architecture）。
- MEng Robotics & Autonomous Systems 官方主题直接覆盖 control、sensing、ML、optimization、embedded / cyber-physical systems 与 capstone。

## 官网核验（2026-07-06）

- 已核验 UCLA Graduate Program Requirements：CS M.S. 有 Plan I thesis 与 Plan II capstone，两者均为 9-course 路径。
- 已核验 Plan I：7 formal courses + 2 courses / 8 units `CS 598` thesis work，且至少 4 门 formal courses 为 200-level CS。
- 已核验 Plan II：至少 5 门 200-level CS；500-level courses 不可计入；capstone 为 individual project + final report + 3-faculty review。
- 已核验 CS breadth requirement：`CS 180`; `111/118/M151B` 选二；`143/161/174A` 选一；`130/131/132` 选一；另需 3 quarters `CS 201` Satisfactory。
- UCLA MEng program overview / autonomous systems 页面本轮可打开，但逐门 concentration course pool 未能稳定从官方页面抓取，需人工复核当年 advising sheet 或 registrar catalog。
