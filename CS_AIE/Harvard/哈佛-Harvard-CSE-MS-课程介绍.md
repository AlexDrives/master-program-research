# 哈佛大学 - Computational Science and Engineering SM 课程介绍

> 来源：Harvard SEAS CSE 官方项目页、Degree Requirements 与 Courses 页面；2026-07-13 核验。  
> 项目：`Master of Science (SM) in Computational Science and Engineering`；Harvard 将该学位简称为 `SM`，本文件以用户表格中的 MS 对应说明。

---

## 项目概况

- CSE 是由 Computer Science、Applied Mathematics 和 Engineering Sciences faculty 共同主导的跨学科项目，重点是数学建模与仿真、并行编程、协作式软件开发，以及大规模数据的组织、可视化、处理和分析。
- SM 为两学期（一学年）项目，须完成 `8` 门课；许多学生延至第三学期以多修课程或安排暑期实习。
- 同一项目也提供两年制、含 thesis 的 `ME`；ME 使用相同的八门课程要求，另有研究学分与论文要求。本文件的主体是 SM。

## 修读要求

| 项目 | 官方要求（SM） |
|---|---|
| 总学分 / 课程 | `32 credits`，`8` 门 letter-graded courses |
| SEAS 课程 | 八门中至少 `4` 门须由 SEAS 提供 |
| Core | `AM 205` 与 `CS 2050` 必修；在 `AM 207` / `AM 215` 中选 `2` 门 |
| 学科选修 | `1` 门 Computer Science elective + `1` 门 Applied Math elective |
| 追加选修 | `2` 门 additional electives；其中第 2 门可为 non-computationally-intensive |
| Research experience | `AC 297r` CSE Capstone Project 完成 |
| 最低成绩 | 单课至少 `C (2.0)`；八门计入课平均至少 `B (3.0)` |
| 课程层级 | 不得计入 `300/3000-level`；最多 `3` 门可为 `100/1000-level SEAS/FAS` 或 MIT U-level；低于此层级及 General Education 均不可计入 |
| Independent study | 最多一学期 `AC 299r` 可计入八门课程 |
| 不接受项目 | Harvard Extension School 课程与 transfer credit 不可计入；requirement waiver 不减少总学分 |

### Core Courses

| 模块 | 官方课程 |

### 学科与追加选修

| 模块 | 官方课程池 / 规则 |
|---|---|
| CS elective | 一门 `COMPSCI` 课程 |
| Applied Math elective | 一门 `APMTH` 课程 |

## 机器人 / 视觉相关性

- 该学位没有单列 robotics 或 computer vision concentration，也没有将机器人/视觉课程列为 core；其直接基础是 `AM 207`（随机方法与 AI）、`AM 205`（数值方法）、`AM 215`（建模）及 `CS 2050`（高性能计算）。
- `APCOMP 209B` 的官方描述明确包含 CNN、RNN、transformers、language models、autoencoders、generative models、Bayesian methods 与 unsupervised learning，适合视觉/感知模型的算法基础，但不是专门的 computer vision 课。
- `APCOMP 215` 关注 AI 系统的 data pipeline、训练、评估、部署、监控、LLM 与 agent-based systems，偏 MLOps/系统落地，而非机器人硬件或控制。
- 机器人、计算机视觉或具身 AI 的直接课程应从 `COMPSCI` / `APMTH` 或获批的 SEAS、Harvard、MIT 选修中安排；是否满足 CS elective、additional elective 与 SEAS 四门课要求，需在实际注册前与项目确认。
