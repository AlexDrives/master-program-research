# 华盛顿大学 - Professional Master's Program (PMP) 课程介绍

> 来源：University of Washington Paul G. Allen School 官方 PMP 项目页、Student Handbook、Courses & Colloquia 页面；2026-07-13 核验。  
> 项目：`Professional Master's Program (PMP), Master of Science`。

---

## 项目概况

- UW Allen School PMP 是为在职软件专业人士设计的 part-time evening professional master's program；项目页称通常需 `40 credit hours`、平均 `2.5-3 years` 完成。
- 该项目面向 core 与 emerging computing areas；官方举例 database management、ML、architecture、computational biology、programming languages、software entrepreneurship、computer vision、ubiquitous computing、robotics 等。
- **它不是 Robotics 或 Perception 专项学位。** 机器人和 computer vision 只是项目页列举的可能主题；没有对应的必修 sequence、specialization 或研究/thesis 路径要求。

## 修读要求

| 模块 | 官方要求 |
|---|---|
| 总学分 | `40 credits`，由 coursework 与 colloquia 组合完成。 |
| PMP coursework | 至少 `32 credits`：完成 `8` 门、每门 `4 credits` 的 `CSE P 5XX` PMP courses。 |
| Non-PMP 替代 | 最多 `2` 门 graduate-level non-`CSE P` 课程可经 approval/petition 替代；通常来自 Allen School daytime graduate program 或相关技术领域。 |
| 剩余 8 credits | 可由 colloquium、额外 coursework 或二者组合完成。`CSE 520` colloquium 最多 `8 credits`（每门 1 credit）可计入。 |
| Course 级别与重修 | 只计 graduate-level（通常 `500+`）课程；`CSE P` 一般不可重复计学分，`CSE P 590 Special Topics` 最多可重复至 `20 credits`（通常相当于 5 门 4-credit sections）。 |
| 课程成绩 | 每门计入学位的 coursework 必须取得 numeric grade `2.7+`；S/NS 通常不计入（官方 COVID extraordinary-circumstances 例外除外）。 |
| Colloquium 成绩 | `CSE 520` 为 CR/NC，必须取得 `CR` 才计入。每注册 1 credit，须观看 8 场并为其中 4 场提交简短报告。 |
| GPA | quarterly 与 cumulative GPA 均须至少 `3.0`；毕业 cumulative GPA 也须 `3.0+`。 |
| 进度 / 年限 | 建议每秋冬春 quarter 1 门 4-credit `CSE P` + 1 credit colloquium，8 quarters 约 2.5 年完成；Graduate School 规定从首次注册起 `6 years` 内完成。 |
| Transfer / GNM | non-`CSE P` 与 GNM/transfer 的组合受 two-course limit 约束；GNM + transfer 合计最多 `12 credits`。另可经 PMP 与 Graduate School 批准，从其他院校转入最多 `6 quarter credits` 的 graduate coursework。 |

## 官方课程结构 / 当前页面列出的课程池

### 固定的课程形式

| 类型 | 官方定义 |
|---|---|
| `CSE P 5XX` | PMP 的 4-credit coursework；8 门构成最低 32 coursework credits。 |
| `CSE P 590` | Special Topics；是唯一可重复计学分的 PMP 课程编号，最多 20 credits。 |
| `CSE 520` | colloquium，1 credit CR/NC；最多计 8 credits。可从 Allen School colloquia、distinguished lectures、research seminars 中选听并报告。 |
| Non-PMP graduate courses | 最多两门、须 petition/approval；课程须 graduate-level。 |

### 机器人 / 感知相关性

- 当前课表最接近机器人大脑的课程是 `CSE P 546 Machine Learning` 和 `CSE P 590 Special Topics: Reinforcement Learning`。
- 当前页面**没有**列出 robotics、computer vision、perception、SLAM、motion planning、control 或 human-robot interaction 课程；`CSE P 557 Computer Graphics` 也不等同于视觉感知。
- 项目总览虽将 computer vision 与 robotics 写作可能主题，但不保证当年开课，也不构成学位方向。数据库、分布式系统、软件测试、创业、量子计算、安全等同样是这个专业型 CS 项目的正式课程生态，不能省略。
