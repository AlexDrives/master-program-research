# 宾夕法尼亚大学 - Computer and Information Science MSE 课程介绍

> 来源：UPenn Graduate Catalog / CIS MSE 官方目录，2026-07-06 核验  
> 包含项目：`M.S.E. in Computer and Information Science`

---

## 项目概况

- CIS MSE 是 Penn Engineering 的计算机与信息科学工程硕士，面向已有较强数学与计算机科学训练的申请者。
- 官方目录说明项目可在 AI、databases、security、programming languages 等方向内做深入选课。
- 本文件仅整理 `CS_AIE/UPenn/宾大-UPenn-CIS-MSE`，不涉及其他学校或其他 UPenn 项目。

## 修读要求

| 项目 | 官方要求 |
|---|---|
| 总学分 | `10 course units` |
| CIS 学分 | 10 个 CU 中至少 `7` 个必须为 CIS 课程 |
| Core | 4 门 core courses；需满足 systems / theory / ML 限制 |
| CIS electives | 3 门 CIS electives |
| Remaining electives | 3 门 CIS 或 approved non-CIS electives |
| GPA | 硕士期间最低 GPA `2.7`；毕业 GPA 也需达到 `2.7` |
| 完成时限 | 最多 7 年完成 MSE |
| Thesis | 可选；`CIS 9990 Master's Thesis Research` 可计 2 CU elective；需 advisor、reader、approval 与 oral presentation |
| Independent study | 最多 2 CU independent study 可用于 CIS MSE electives |

## 官方课程结构 / 全部课程池

### Core Courses（4 门）

官方要求 4 门 core 中必须包含：至少 1 门 systems course 或 `CIS 5010`；至少 1 门 theory course；最多 1 门 machine-learning course 计入 core（其他 ML 课仍可作 elective）。

| 类别 | 官方课程池 |
|---|---|
| Theory Courses | `CIS 5020 Analysis of Algorithms`; `CIS 5110 Theory of Computation` |
| Systems Courses | `CIS 5050 Software Systems`; `CIS 5480 Operating Systems Design and Implementation`; `CIS 5530 Networked Systems`; `CIS 5550 Internet and Web Systems` |
| Machine-Learning Courses | `CIS 5200 Machine Learning`; `CIS 5190 Applied Machine Learning`; `CIS 5210 Artificial Intelligence` |
| Other Courses | `CIS 5000 Software Foundations`; `CIS 5710 Computer Organization and Design` |

### CIS Elective Courses

- 任意 `CIS 5000` 到 `CIS 7000` 编号课程可作为 CIS electives。
- 最多 1 门 `CIS 7000` class 可计入。

### CIS or Non-CIS Elective Courses

- 剩余 3 门课程可为 `CIS 5000` 到 `CIS 8000` 编号课程，或官方 approved non-CIS courses。
- Thesis 路径中 `CIS 9990` 可作为 2 CU elective。

## 官方课程描述抓手

- 官方 AI concentration 明确包含 ML、AI、deep learning、NLP、machine perception、computer vision、mechatronics 和 robotics 课程。
- 官方 Computer Vision concentration 是最直接的感知方向路径：`CIS 5200 / 5800 / 5810 / 6800`。
- 机器人/感知相关抓手：`CIS 5800`, `CIS 5810`, `CIS 6800`, `MEAM 5100`, `MEAM 5200`, `ESE 5460`。
- 与机器人无关但属于官方结构的 theory、systems、software foundations、security/databases 课程已完整保留。
