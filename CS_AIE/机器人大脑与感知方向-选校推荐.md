|---|---|---|

## 最终推荐顺序

如果按“专业排名优先，但允许 robotics track 和课程相关度上调/下调”的综合结果，我建议：

1. CMU MRSD / MSCV / MS Robotics
2. UCSD CSE MS
3. Georgia Tech MSCS - Computational Perception and Robotics
4. Northeastern MS AI
5. USC MS CS-AI
6. Columbia MS CS VGIR 或 MS AI Robotics and Perception
7. JHU CS MSE AI Track
8. UCI MCS

备选：Purdue MS CS、Cornell Ithaca MEng CS、UPenn CIS MSE、UCLA MEng Robotics。  
不建议作为主推荐：NYU Courant MSCS、Maryland MS CS、Northwestern MS CS、UCSB MS CS；Stanford / Berkeley 可作为彩票冲刺，不应挤占主力名额。

## 文书主线建议

- 主线建议写：`robot perception and decision-making for embodied autonomy`。
- 机器狗 RL navigation 对应 robotics、planning、reinforcement learning、control、robot autonomy。
- Affordance / environment reasoning 视觉模型对应 computer vision、machine perception、embodied AI、vision-language/environment reasoning。
- 对 CMU / UCSD / Georgia Tech：强调 perception 到 decision-making 的闭环。
- 对 Columbia / JHU：强调 vision/perception 与 robotics track 的结合。
- 对 USC / Northeastern / UCI：强调 AI systems、vision、project-based learning 和工程落地。
