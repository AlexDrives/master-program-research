# 机器人大脑 + 感知 匹配标准

## 机器人大脑 (Robot Brain)
规划、决策、学习、控制——机器人的"思考"部分

核心关键词：
- Robot Planning / Motion Planning / Path Planning
- Robot Learning / Learning for Robotics
- Reinforcement Learning (尤其 applied to robotics)
- Robot Control / Optimal Control / Model Predictive Control
- Embodied AI / Embodied Agents
- SLAM / Localization / Mapping
- Robot Autonomy / Autonomous Systems
- Sensorimotor Learning
- Multi-Agent / Distributed Robotics
- Robot Decision Making

## 感知 (Perception)
让机器人"看见"和"理解"世界

核心关键词：
- Computer Vision / Machine Vision
- Machine Perception / Robot Perception
- 3D Reconstruction / 3D Vision / Depth Estimation
- Image Processing / Digital Image Processing
- Deep Learning for Vision (CNN, ViT, etc.)
- Sensor Fusion / Multi-Sensor Integration
- Visual SLAM / Visual Odometry
- Object Detection / Segmentation / Recognition
- Point Cloud / LiDAR Processing
- Computational Imaging / Computational Photography
- Signal Processing (foundational)
