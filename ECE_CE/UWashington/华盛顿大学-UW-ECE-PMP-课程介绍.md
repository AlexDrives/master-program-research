# 华盛顿大学 — ECE 硕士项目课程介绍

> 来源：UW ECE 官网 `Master's & Ph.D. Programs`、`Degree Requirements — MSEE`、`Professional Master's Program`、官方课程目录，2026-07-02 核验
> 项目：`Professional Master's Program in ECE (PMP)`

---

## 项目概况

- **项目性质**：`professionally-oriented graduate degree`
- **课程风格**：`industry-driven curriculum`
- **授课者**：`UW ECE faculty and industry leaders`
- **时间安排**：官网强调 evening courses，适合在职与全日制学生结合 internships/career development

## 修读要求

- **总学分**：`45 credits`
- **构成**：由 `coursework credits + colloquium seminar credits` 组成
- **全日制节奏**：通常 `4-5 quarters`（约 1-1.5 年）；每 quarter 至少 `10 credits`，整个项目期间至少 `5 seminar credits`
- **兼读节奏**：通常 `9 academic quarters`（约 3 年）；常见安排为每 quarter `1 门 4-credit course + 1 credit seminar`
- **论文**：PMP 是 coursework-focused degree，不以 thesis / PhD preparation 为主要目标；若想做 thesis 或继续 PhD，官网建议看 daytime graduate programs

## 官方课程结构

### 项目风格信号

官网主页面对它的定位非常明确：

- flexible degree
- industry-responsive curriculum
- experiential learning and research opportunities

这说明 UW 的 PMP 更像“工程实践导向的 ECE 硕士”，而不是研究型 thesis master。

### 官网可见课程线索

这次从 UW 官方课程目录里，已经可以把 PMP 可能依托的 `signal / ML / control / robotics / embedded` 课程链条看得更清楚：

| 课程代码 | 英文课程名 | 中文理解 | 说明 |
|------|------|------|------|
| `E E 443` | Machine Learning for Signal Processing Applications | 面向信号处理应用的机器学习 | 直接提到 signal / image / video processing、CNN、object detection、segmentation |
| `E E 445` | Fundamentals of Optimization and Machine Learning | 优化与机器学习基础 | convex optimization、regularized regression/classification |
| `E E 446` | Tiny Machine Learning for Ultra Low-Power Edge Computing | 超低功耗边缘计算 TinyML | ML on microcontrollers / embedded sensors |
| `E E 447` | Control System Analysis I | 控制系统分析 I | 经典反馈控制与 state variable methods |
| `E E 449` | Robotics and Controls Capstone | 机器人与控制毕业设计 | sensing、actuation、feedback loop 的 team-based robotics design |
| `E E 513` | Introduction to Deep Learning Applications in Engineering | 工程深度学习应用导论 | neural networks、generative networks、transformers、reinforcement learning |
| `E E 518` | Digital Signal Processing | 数字信号处理 | multirate、filter banks、machine learning applications |
| `E E 540` | Wearable Robotics | 可穿戴机器人 | design / control / evaluation of wearable robots |
| `E E 544` | Computer Systems Architecture | 计算机系统体系结构 | processor design、memory hierarchy、multiprocessors |
| `E E 547` | Linear Systems Theory | 线性系统理论 | controllability、observability、realization theory |
| `E E 585` | Learning-Based Control | 基于学习的控制 | learning / optimization of controllers、reinforcement learning |
| `E E 594` | Robust Control | 鲁棒控制 | H-infinity/H2 control、uncertain systems、robust synthesis |

这组课非常关键，因为它说明 UW PMP 虽然项目页本身是 `professionally-oriented` 包装，但底层可挂靠的官方课程池并不弱，尤其在 `signal processing + machine learning + control + robotics + embedded systems` 交叉地带。

## 官方课程描述抓手

- `E E 443`：Machine Learning for Signal Processing Applications，课程目录提到 signal / image / video processing、CNN、object detection、segmentation。
- `E E 446`：Tiny Machine Learning for Ultra Low-Power Edge Computing，面向 microcontrollers / embedded sensors。
- `E E 445 / 513`：优化、机器学习和深度学习工程应用。
- `E E 449`：Robotics and Controls Capstone，覆盖 sensing、actuation、feedback loop 的 team-based robotics design。
- `E E 518`：Digital Signal Processing，覆盖 multirate、filter banks、machine learning applications。
- `E E 540`：Wearable Robotics。
- `E E 544`：Computer Systems Architecture。
- `E E 547`：Linear Systems Theory。
- `E E 585 / 594`：Learning-Based Control 与 Robust Control。
