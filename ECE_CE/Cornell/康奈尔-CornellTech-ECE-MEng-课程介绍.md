# 康奈尔大学 Cornell Tech — 电气与计算机工程硕士 (Cornell Tech ECE M.Eng.) 课程介绍

> 来源：Cornell Tech 官方项目页、Cornell Tech 招生页、Cornell Duffield ECE 页面，2026-07-02 核验
> 项目：`Master in Electrical and Computer Engineering`

---

## 项目概况

Cornell Tech 这条线需要和 Ithaca 的传统 ECE M.Eng. 分开看。

- 官方项目页直接把它写成：
  - `flexible engineering master's degree`
  - 可去 `prototype new hardware`
  - 可去 `improve advanced algorithms`
- Cornell Duffield ECE 页面进一步把它概括成：
  - `product-oriented`
  - `studio-driven`
  - in `New York City`
  - focused on `data, systems, and innovation`

这说明 Cornell Tech ECE 不是 Ithaca ECE M.Eng. 的地点变体，而是一条培养目标明显不同的、偏产品化和创业导向的 ECE 硕士路线。

---

## 修读要求

### 总体结构

- **总学分**：标准 `30 credits`
- **学制**：标准 `1 year`
- **每学期**：最低 `12 credits`
- **建议负载**：通常每学期 `15 credits`
- **上限**：每学期 `18 credits`
- **评分要求**：
  - 累计 `GPA >= 2.50`
  - 每门课通常需 `C- or better`
  - `Studio` 课程需 `B or above`
- **评分方式限制**：
  - 最多 `2 credits S/U`
  - 至少 `28 / 30 credits` 为字母评分

### 30 学分结构（18 + 8 + 4）

| 模块 | 学分 | 说明 |
|------|------|------|
| 技术课 | `18` | 含技术必修 + 技术选修 |
| Studio 实践 | `8` | 秋季 Product Studio + 春季三选一 Studio + 1 学分 studio 选修 |
| 通用选修 | `4` | 可来自 ECE / CS / ORIE / INFO / DESIGN / LAW / NBAY / TECH 等 |

## 官方课程描述抓手

### 第一门技术必修（2 选 1）

| 代码 | 英文名称 | 中文理解 |
|------|----------|----------|
| `ECE 5414` | Applied Machine Learning | 应用机器学习 |
| `CS 5781` | Machine Learning Engineering | 机器学习工程 |

### 第二门技术必修（4 选 1）

| 代码 | 英文名称 | 中文理解 |
|------|----------|----------|
| `ECE 5110` | Random Signals in Communications and Signal Processing | 通信与信号处理中的随机信号 |
| `ECE 5415` | Applied Digital Signal Processing / DSP and Learning | 应用数字信号处理 / DSP 与学习 |
| `ECE 5746` | Applied Digital ASIC Design | 应用数字 ASIC 设计 |
| `ECE 5755` | Modern Computer Systems and Architecture | 现代计算机系统与体系结构 |

### 课程结构意味着什么

这组必修很有代表性，因为它直接说明 Cornell Tech ECE 至少会从下面几条线里强制占两条：

- `machine learning`
- `signal processing`
- `digital ASIC / hardware`
- `computer systems and architecture`

也就是说，它不是单纯靠 studio 项目包装的“泛科技硕士”，而是确实把 `ML + signal + hardware + systems` 放进了 ECE 课程骨架里。

## 官方课程结构

Cornell Tech 这条线最像它自己的地方，不是普通技术选修，而是 studio 结构。

| 代码 | 英文名称 | 中文理解 | 学分 |
|------|----------|----------|------|
| `TECH 5900` | Product Studio | 产品工作室 | `4` |
| `TECH 5910` | Startup Studio | 创业工作室 | `3` |
| `TECH 5920` | BigCo Studio | 大企业合作工作室 | `3` |
| `TECH 5930` | PiTech Impact Studio | 公益技术影响力工作室 | `3` |

规则上通常是：

- 秋季修 `TECH 5900 Product Studio`
- 春季在 `Startup / BigCo / PiTech` 中三选一
- 再加 `1 credit` studio elective

这让它和 Ithaca 的 `ECE 5995/5996 design project` 路线明显不同。Cornell Tech 更偏：

- 跨学科团队
- 产品落地
- 创业/产业协作
- NYC tech ecosystem

## 项目概况补充

这条线在招生层面也是独立项目，不只是校区差异：

- Cornell Tech 招生页明确说明：`Master of Engineering` 申请要走 Cornell Tech 自己的申请流程
- FAQ 明确说明：每个申请季在 Cornell Tech 只能申请 `one master's program`
- FAQ 还明确说明：Cornell Tech 与 Ithaca campus 使用 `separate application system`

这进一步说明：

- `Cornell Tech ECE M.Eng.` 应被视为一条独立申请标的
- 不应只附着在 Ithaca ECE 文件里带过
