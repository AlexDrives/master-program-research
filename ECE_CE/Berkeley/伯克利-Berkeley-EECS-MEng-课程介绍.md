## 修读要求

### 基本概况
- **总学分**：25 学分
- **学制**：1 学年（Fall + Spring，含两个 Boot Camp 密集课程），9 个月
- **GPA 要求**：累计 GPA 不低于 3.0，单科不低于 C-
- **学位性质**：终端专业学位（非博士跳板），STEM 认证
- **GRE**：不再要求/接受

### 三大课程模块

**1. 核心技术课程 -- 12 学分**
- 4 门 EECS 研究生 200 级课程（每学期 2 门）
- 至少 2 门来自所选 Concentration 的 Core 列表
- 至少 3 门来自 Core + Expanded 列表
- 第 4 门可从本专业或其他 MEng Concentration 中选
- 全部 Letter Grade

**2. 领导力课程 (Leadership) -- 8 学分**
- 秋季 Boot Camp（8 月中旬，2 学分）：ENGIN 270A + 270B
- 秋季学期（2 学分）：ENGIN 270C + ENGIN 295（个体沟通）
- 春季 Boot Camp（1 月上旬，2 学分）：从 270D/G/H/I/J/L/M/N/P/Q/R 中任选 2 门
- 春季学期（2 学分）：ENGIN 270K + ENGIN 295（团队沟通）

**3. Capstone 项目 -- 5 学分**
- ENGIN 296MA（秋季，2 学分）+ ENGIN 296MB（春季，3 学分）
- 3-5 人团队，可选 Faculty Project 或 Industry Partner Project
- 春季末交最终报告+展示

### 综合考试
- **领导力考试**：Fung Institute 组织笔试（考察 ENGIN 270A/B/C + 295）
- **技术考试**：EECS 系组织（可能口试或笔试）

### 6 个技术方向 (Concentrations)
1. Data Science（数据科学）
2. Computer Systems（计算机系统）
3. Physical Electronics and Integrated Circuits（物理电子学与集成电路）
4. Robotics and Embedded Software（机器人与嵌入式软件）
5. Signal Processing and Communications（信号处理与通信）
6. Visual Computing and Computer Graphics（视觉计算与计算机图形学）

## 官网核验（2026-07-02）

- 已核验项目页：https://eecs.berkeley.edu/academics/graduate/industry-programs/meng/
- Berkeley 官网确认：MEng 是 `accelerated program`，目标是培养进入工程行业的未来领导者，核心结构是 `technical specialization + engineering leadership + team-oriented capstone project`。
- 官网同时强调，EECS 的技术课程按 `technical concentrations` 组织，真正的课程细目需进一步进入 `EE Courses / CS Courses` 页面查看。
- 项目页正文还明确写到：Berkeley MEng 不只是“多上几门课”，而是把 `deeper technical specialization`、`engineering leadership` 和 `team-oriented capstone` 绑定成一个完整培养模型，和传统 course-heavy MS 区别很大。
- 官网还公开了当年 capstone project marketplace 入口，这再次说明 Berkeley 的强项在于“高密度核心课 + 强项目制落地”，而不是像 Stanford / UPenn Robotics 那样靠长课程链铺开机器人知识图谱。
- 同时核验 MEng requirements 页：https://eecs.berkeley.edu/book/meng-guide/requirements/
- 该页进一步把技术课要求明确为：`4 门 graduate-level EECS technical courses`，其中 `2 门必须来自 admitted concentration 的 core list`，且 `4 门里至少 3 门` 要来自 `core + expanded list`。这也解释了 Berkeley 为什么虽然有强课，但一年制下真正可自由试错的技术课位子并不多。
- 因此，Berkeley 的项目主页更适合确认“培养模式和项目定位”，而具体课程深度仍需配合课程目录和 concentration 页面使用。
