# 波士顿大学 — ECE / Robotics 硕士项目课程介绍

> 来源：Boston University ENG 官网、Academic Bulletin，2026-07-02 核验
> 项目：`MS in Robotics & Autonomous Systems`

---

## 项目概况

- BU 官方项目页将其设置为 `MS in Robotics & Autonomous Systems`。
- 项目与 product design、healthcare、medicine、autonomy 等应用场景相连。
- 课程结构由导论课、四类核心桶、technical electives，以及 thesis/non-thesis 路径组成。

## 修读要求

- **总学分**：`32 graduate-level units`
- **课程层级**：`500-level or above`
- **成绩要求**：课程计入要求需 `C or better`
- **最低 GPA**：`3.0`
- **完成时间**：通常 `2-3 semesters`

## 官方课程结构

| 模块 | 要求 |
|------|------|
| Introduction to Robotics & Autonomous Systems | `4 credits` |
| Robotics Core Courses | `16 credits` |
| Technical Electives | `12 credits` |

### Robotics Core 的公开逻辑

官网与 handbook 都把 16 学分核心课按四类组织，每类至少 1 门：

- `Control (C)`
- `Design (D)`
- `Perception (P)`
- `Machine Learning (ML)`

这点非常重要，因为它说明 BU Robotics 的课程结构天然就是按“机器人脑 + 感知 + 机械/控制 + 学习”四块拼起来的。

### 官网已能确认的代表性课程

BU 官方项目页和 bulletin 已经能落到以下课程安排：

| 模块 | 英文课程名 | 中文理解 | 类别 |
|------|------|------|------|
| Required | Introduction to Robotics | 机器人导论 | Required Course |
| Robotics Core | Robot Motion Planning | 机器人运动规划 | Control / Perception |
| Robotics Core | Medical Robotics | 医疗机器人 | Design |
| Robotics Core | Vision, Robotics, and Planning | 视觉、机器人与规划 | Control / Perception |
| Robotics Core | Soft Robotics | 软体机器人 | Design |
| Robotics Core | Formal Methods in Robotics | 机器人形式化方法 | Control |
| Robotics Core | Cyberphysical Systems | 信息物理系统 | Perception |
| Robotics Core | Learning from Data | 从数据中学习 | Machine Learning |
| Robotics Core | Machine Learning | 机器学习 | Machine Learning |

BU 官方对 `ENG EK 505` 的课程说明已经写得比较具体：

- 覆盖 `kinematics and dynamics`
- 涉及 `control and motion planning`
- 涉及 `sensing and perception`
- 还会讨论 `machine learning techniques in robotics`

再结合 handbook 里对 `Control / Design / Perception / Machine Learning` 四类核心桶的要求，可以确认 BU 的项目不是“只有一门导论课撑场面”，而是导论课先打总基础，再用四个核心桶补足机器人结构。

### Thesis / Non-thesis 两条路径

新版 handbook 还把 thesis 和 non-thesis 的差异单独写清楚了：

- `non-thesis option`
  - `1` 门导论课
  - `4` 门核心桶课程
  - `3` 门 technical electives
- `thesis option`
  - `1` 门导论课
  - `4` 门核心桶课程
  - `1` 门 technical elective
  - `2` 门 `MS thesis courses (ME 954)`

这意味着 BU 不只是“课程结构概念上很漂亮”，连研究型与非研究型路径都公开得相当清楚。

## 官方课程描述抓手

- `ENG EK 505`：覆盖 kinematics and dynamics、control and motion planning、sensing and perception、machine learning techniques in robotics。
- `Robot Motion Planning`：对应机器人路径/运动规划，属于 Control / Perception 核心桶。
- `Vision, Robotics, and Planning`：把视觉、机器人和规划合在一起，属于 Control / Perception 核心桶。
- `Formal Methods in Robotics`：对应机器人系统形式化方法，属于 Control 核心桶。
- `Cyberphysical Systems`：对应 CPS 与感知/系统耦合，属于 Perception 核心桶。
- `Learning from Data / Machine Learning`：对应 Machine Learning 核心桶。
- `Medical Robotics / Soft Robotics`：对应机器人设计与应用系统，属于 Design 核心桶。
- Core buckets 按 `Control / Design / Perception / Machine Learning` 四类组织，每类至少一门。
- Thesis option 使用 `ME 954` thesis courses；non-thesis option 使用 technical electives 补足剩余学分。
