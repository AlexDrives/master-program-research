# 卡内基梅隆 CMU - ECE MS 课程介绍

> 来源：CMU ECE 官方 `Requirements / Standard Program` 页面、CMU Robotics Institute 官方课程与研究页面、CMU Machine Learning 官方页面（2026-07-02 核验）
> 学位：Master of Science in Electrical and Computer Engineering

---

## 项目概况

- CMU ECE `MS in ECE` 是 full-time professional master's。
- `Standard program` 通常为 `3 semesters`。
- 另有更长的 `Advanced Study` 路线

## 修读要求

### Standard Program

- 总计 `97 units`
- 必修 `18-989 Introduction to Graduate Studies`，`1 unit`
- 主体课程通常为 `600-level and above`
- 最多 `27 units` 的 `research project coursework` 可计入学位要求

### 97 units 结构

| 模块 | 要求 |
|------|------|
| ECE Core | `60 units` |
| CIT Electives | `24 units` |
| General Technical Electives | `12 units` |
| Introduction to Graduate Studies | `1 unit` |

### Intensive Project Option

- 可走更强项目/研究导向的 `Intensive Project Option`
- 官方页面明确给出相应 research-intensive 结构

## 官方课程描述抓手

### 16-385 Computer Vision

CMU 官方课程站明确写到它覆盖：

- `image processing`
- `detection and recognition`
- `geometry-based vision`
- `physics-based vision`
- `video analysis`

这是一门很标准、也很强的计算机视觉主线课。

### 16-720 / 16-822 / 16-824 Vision Courses

RI / vision@CMU 课程生态中还可见 `16-720 Computer Vision`、`16-822 Geometry-based Methods in Vision`、`16-824 Visual Learning and Recognition` 等视觉课程，说明 CMU 的视觉链条不是单点课程，而是有几何视觉、视觉学习、识别等多个层级。

### 16-831 Introduction to Robot Learning

RI 官方 curriculum 页面明确列出了这门课。

它的存在很关键，因为这说明 CMU ECE 学生若往 RI 跨选，不只是“能学 vision”，还能直接碰到 robot learning 主线。

### 16-833 Robot Localization and Mapping

RI 官方 curriculum 页面和 CMU 官方师资材料都能稳定看到这门课。

这意味着 `localization / mapping / state estimation` 这种非常贴机器人感知的问题，在 CMU 是有正式课程抓手的。

### 16-811 Math Fundamentals for Robotics

RI curriculum 中可见 `Math Fundamentals for Robotics`，可作为机器人建模、状态估计、规划和控制课程的数学入口。

### 10-702 Statistical Machine Learning

CMU 机器学习/统计官方页面能稳定挂到这门高阶机器学习课。

这让 CMU ECE 的算法侧不是抽象“学校 AI 很强”，而是确实可以借到体系化的统计机器学习课程。
